## 1. 모수 vs 비모수 검정

### 1.1 비교표

| 특성 | 모수 검정 (Parametric) | 비모수 검정 (Nonparametric) |
|------|----------------------|---------------------------|
| **분포 가정** | 정규분포 필요 | 분포 가정 없음 |
| **데이터 타입** | 연속형 (등간/비율 척도) | 서열형, 연속형 모두 가능 |
| **이상치 민감도** | ✅ 민감 (평균 사용) | ❌ 강건 (순위/중위수 사용) |
| **검정력** | ✅ 높음 (가정 충족 시) | ⚠️ 약간 낮음 (5-10% 손실) |
| **샘플 크기** | 큰 샘플에서 더 정확 | 작은 샘플에서도 강건 |
| **해석** | 평균 차이 | 중위수/분포 차이 |

---

### 1.2 대응 관계

| 연구 목적 | 모수 검정 | 비모수 대안 | 검색 키워드 |
|----------|----------|------------|------------|
| **두 그룹 비교 (독립)** | 독립표본 t-검정 | Mann-Whitney U | 't-test', 'mann-whitney' |
| **두 그룹 비교 (쌍)** | 대응표본 t-검정 | Wilcoxon Signed-Rank | 'paired-t', 'wilcoxon' |
| **세 그룹 이상 (독립)** | 일원 ANOVA | Kruskal-Wallis | 'anova', 'kruskal-wallis' |
| **세 그룹 이상 (쌍)** | 반복측정 ANOVA | Friedman 검정 | 'repeated-measures', 'friedman' |
| **상관분석** | Pearson r | Spearman ρ, Kendall τ | 'correlation', 'spearman' |

---

### 1.3 선택 기준

```
정규성 검정 (Shapiro-Wilk)
│
├─ p > .05 (정규분포) + n ≥ 30
│   └─ 모수 검정 선택 ⭐ (검정력 높음)
│
├─ p > .05 (정규분포) + n < 30
│   └─ 모수 또는 비모수 (둘 다 가능)
│
└─ p ≤ .05 (정규성 위반) 또는 서열 데이터
    └─ 비모수 검정 선택 ⭐ (가정 불필요)
```

**권장**: 의심스러우면 비모수 검정 사용 (더 안전)

## 2. t-test 계열 비교

### 2.1 세 가지 t-test

| 검정 | 데이터 구조 | 가정 | 사용 예 | 검색어 |
|------|------------|------|---------|--------|
| **독립표본 t-검정** | 두 독립 그룹 | 정규성, 등분산 | 실험군 vs 대조군 | 't-test' |
| **Welch's t-test** | 두 독립 그룹 | 정규성 (등분산 불필요) | 샘플 크기 다른 그룹 | 'welch' |
| **대응표본 t-검정** | 쌍 데이터 (Before-After) | 차이값 정규성 | 전후 비교, 쌍둥이 | 'paired-t' |

---

### 2.2 독립표본 vs Welch's t-test

**공통점**:
- 두 독립 그룹 평균 비교
- 정규성 가정 필요

**차이점**:

| 특성 | 독립표본 t-검정 | Welch's t-test |
|------|---------------|----------------|
| **등분산 가정** | ✅ 필요 (Levene p > .05) | ❌ 불필요 |
| **자유도 (df)** | n₁ + n₂ - 2 | Welch-Satterthwaite 공식 |
| **언제 사용** | 등분산 + 유사 샘플 크기 | 등분산 위반 또는 샘플 크기 차이 |
| **검정력** | 등분산 시 높음 | 등분산 위반 시 더 정확 |

**의사결정**:
```
Levene's Test
│
├─ p > .05 (등분산) → 독립표본 t-검정
└─ p ≤ .05 (등분산 위반) → Welch's t-test ⭐
```

---

### 2.3 독립표본 vs 대응표본 t-검정

**핵심 차이: 데이터 독립성**

| 특성 | 독립표본 | 대응표본 |
|------|---------|---------|
| **샘플 관계** | 서로 독립 | 쌍으로 연결 (paired) |
| **비교 대상** | 두 그룹 평균 | 쌍 차이의 평균 |
| **검정력** | 낮음 (개인차 포함) | ✅ 높음 (개인차 제거) |
| **예시** | 남성 vs 여성 | 전 vs 후, 좌 vs 우 |

**주의**: 데이터 구조 잘못 선택 시 Type I Error 증가

---

### 2.4 t-test vs Mann-Whitney U

| 특성 | t-test | Mann-Whitney U |
|------|--------|----------------|
| **가정** | 정규성, 등분산 | 분포 가정 없음 |
| **사용 데이터** | 원본 값 (평균) | 순위 (rank) |
| **이상치 영향** | ✅ 민감 | ❌ 강건 |
| **해석** | "평균 차이" | "분포 차이 (중위수 또는 순위)" |
| **검정력** | 높음 (정규분포 시) | 5% 손실 (정규분포 시) |

**선택 기준**:
- 정규성 충족 + 이상치 없음 → t-test
- 정규성 위반 또는 이상치 존재 → Mann-Whitney U

## 3. ANOVA 계열 비교

### 3.1 ANOVA 종류

| ANOVA 타입 | 독립변수 개수 | 반복측정 | 공변량 | 검색어 |
|-----------|-------------|---------|-------|--------|
| **일원 ANOVA** | 1개 | ❌ | ❌ | 'anova', 'one-way' |
| **이원 ANOVA** | 2개 | ❌ | ❌ | 'two-way-anova' |
| **삼원 ANOVA** | 3개 | ❌ | ❌ | 'three-way-anova' |
| **반복측정 ANOVA** | 1개 (시간) | ✅ | ❌ | 'repeated-measures' |
| **혼합 ANOVA** | 1개 Between + 1개 Within | ✅ | ❌ | 'mixed-anova' |
| **ANCOVA** | 1개 | ❌ | ✅ | 'ancova' |
| **MANOVA** | 1개, 종속변수 여러 개 | ❌ | ❌ | 'manova' |

---

### 3.2 일원 ANOVA vs Kruskal-Wallis

| 특성 | 일원 ANOVA | Kruskal-Wallis |
|------|-----------|----------------|
| **가정** | 정규성, 등분산 | 분포 가정 없음 |
| **사용 데이터** | 원본 값 | 순위 |
| **사후 검정** | Tukey HSD, Bonferroni | Dunn's test |
| **검정력** | 높음 (가정 충족 시) | 약간 낮음 |
| **해석** | "그룹 간 평균 차이" | "그룹 간 중위수/분포 차이" |

---

### 3.3 일원 ANOVA vs 이원 ANOVA

| 특성 | 일원 ANOVA | 이원 ANOVA |
|------|-----------|-----------|
| **독립변수** | 1개 (예: 약물 종류) | 2개 (예: 약물 + 성별) |
| **주효과** | 1개 | 2개 (Main Effect A, B) |
| **상호작용** | ❌ | ✅ Interaction (A × B) |
| **질문** | "약물 효과 있나?" | "약물 효과는? 성별 효과는? 약물×성별 상호작용은?" |

**상호작용 예시**:
```
약물 A: 남성에게는 효과 큼, 여성에게는 효과 작음
→ 상호작용 유의 (F_interaction, p < .05)
→ 단순 주효과 분석 필요
```

---

### 3.4 반복측정 ANOVA vs Friedman

| 특성 | 반복측정 ANOVA | Friedman 검정 |
|------|---------------|--------------|
| **가정** | 정규성, 구형성 | 분포 가정 없음 |
| **데이터 구조** | 동일 개체의 여러 측정 | 동일 개체의 여러 측정 |
| **시점** | 3개 이상 | 3개 이상 |
| **사후 검정** | Bonferroni | Nemenyi, Conover |
| **예시** | 약물 투여 전-중-후 혈압 | 약물 투여 전-중-후 통증 점수 (서열) |

---

### 3.5 ANOVA vs ANCOVA

| 특성 | ANOVA | ANCOVA |
|------|-------|--------|
| **공변량 (Covariate)** | ❌ 없음 | ✅ 있음 (연속형 통제 변수) |
| **목적** | 그룹 간 평균 차이 | 공변량 통제 후 그룹 차이 |
| **검정력** | 낮음 (분산 큼) | ✅ 높음 (공변량 제거로 오차 감소) |
| **예시** | 교수법 A vs B → 성적 | 교수법 A vs B → 성적 (사전 점수 통제) |

**ANCOVA 사용 시기**:
- 사전 점수(pretest) 통제
- 혼란 변수(confounding variable) 통제
- 준실험 설계 (무작위 배정 실패)

---

### 3.6 ANOVA vs MANOVA

| 특성 | ANOVA | MANOVA |
|------|-------|--------|
| **종속변수** | 1개 | ✅ 2개 이상 (다변량) |
| **Type I Error** | 여러 ANOVA 반복 시 증가 | ✅ 단일 검정으로 제어 |
| **예시** | 교수법 → 수학 점수 | 교수법 → 수학 점수 + 과학 점수 + 태도 |

**MANOVA 사용 시기**:
- 종속변수 간 상관 높음 (측정이 관련됨)
- 다중 검정 문제 (multiple testing) 방지

---

## 4. 상관분석 비교

### 4.1 세 가지 상관계수

| 상관계수 | 데이터 요구사항 | 관계 타입 | 해석 | 검색어 |
|---------|---------------|----------|------|--------|
| **Pearson r** | 양 변수 정규분포 | 선형 관계 | 선형 상관 강도 | 'pearson', 'correlation' |
| **Spearman ρ** | 서열 데이터 가능 | 단조 관계 (monotonic) | 순위 상관 | 'spearman' |
| **Kendall τ** | 서열 데이터 가능 | 단조 관계 | 일치/불일치 쌍 비율 | 'kendall' |

---

### 4.2 Pearson vs Spearman

**Pearson r**:
```
r = Cov(X, Y) / (SD_X × SD_Y)

특징:
- ✅ 정규분포 시 최적
- ✅ 해석 명확 (r² = 공유 분산)
- ❌ 이상치에 민감
- ❌ 비선형 관계 탐지 못함
```

**Spearman ρ**:
```
ρ = Pearson r of ranks

특징:
- ✅ 정규성 가정 불필요
- ✅ 단조 비선형 관계 탐지 (예: 지수 관계)
- ✅ 이상치에 강건
- ❌ r²같은 직관적 해석 부족
```

**예시**:
```
데이터: X = [1, 2, 3, 10, 20], Y = [2, 4, 6, 8, 10]

Pearson r = 0.89 (이상치 10, 20에 영향)
Spearman ρ = 1.00 (완벽한 단조 증가)
```

---

### 4.3 Spearman vs Kendall

| 특성 | Spearman ρ | Kendall τ |
|------|-----------|-----------|
| **계산 방법** | 순위의 Pearson 상관 | 일치 쌍 / 전체 쌍 |
| **범위** | -1 ~ +1 | -1 ~ +1 |
| **해석** | 순위 선형 관계 | 일치/불일치 확률 |
| **동점 처리** | 문제 발생 가능 | ✅ 동점 보정 공식 |
| **작은 샘플** | 검정력 높음 | 검정력 약간 낮음 |
| **큰 샘플** | 유사 | 유사 |

**선택 기준**:
- 일반적인 경우 → Spearman ρ (더 직관적)
- 동점(tie) 많은 경우 → Kendall τ (보정 정확)

---

### 4.4 상관분석 vs 회귀분석

| 특성 | 상관분석 | 회귀분석 |
|------|---------|---------|
| **목적** | 관계의 강도/방향 | 예측/인과 관계 |
| **독립/종속변수** | 구분 없음 (대칭) | ✅ 구분 (Y ← X) |
| **예측** | ❌ 불가능 | ✅ Y = β₀ + β₁X |
| **해석** | "X와 Y는 상관" | "X가 1 증가 시 Y는 β₁ 증가" |
| **다변량** | Partial correlation | Multiple regression |

## 5. 회귀분석 비교

### 5.1 회귀 종류

| 회귀 타입 | 종속변수 타입 | 독립변수 개수 | 예시 | 검색어 |
|----------|-------------|-------------|------|--------|
| **단순 선형 회귀** | 연속형 | 1개 | 공부 시간 → 성적 | 'regression', 'simple' |
| **다중 선형 회귀** | 연속형 | 2개 이상 | 공부 시간 + IQ → 성적 | 'multiple-regression' |
| **로지스틱 회귀** | 이진 범주 (0/1) | 1개 이상 | 흡연 + 운동 → 질병 (유/무) | 'logistic' |
| **다항 로지스틱** | 범주 (3개 이상) | 1개 이상 | 성별 + 나이 → 직업 (A/B/C) | 'multinomial-logistic' |
| **순서 로지스틱** | 서열 범주 | 1개 이상 | 교육 수준 → 만족도 (낮/중/높) | 'ordinal-regression' |
| **포아송 회귀** | 카운트 (0, 1, 2...) | 1개 이상 | 인구 밀도 → 범죄 건수 | 'poisson' |

---

### 5.2 단순 vs 다중 선형 회귀

**단순 선형 회귀**:
```
Y = β₀ + β₁X + ε

예: 성적 = 50 + 5×(공부 시간) + ε

해석: "공부 1시간 증가 시 성적 5점 증가"
```

**다중 선형 회귀**:
```
Y = β₀ + β₁X₁ + β₂X₂ + ε

예: 성적 = 30 + 4×(공부 시간) + 0.5×(IQ) + ε

해석: "IQ 고정 시, 공부 1시간 증가 → 성적 4점 증가"
```

**다중공선성 주의**:
- X₁과 X₂가 높은 상관 → 계수 불안정
- VIF > 10 → 변수 제거 또는 Ridge/Lasso

---

### 5.3 선형 회귀 vs 로지스틱 회귀

| 특성 | 선형 회귀 | 로지스틱 회귀 |
|------|----------|--------------|
| **종속변수** | 연속형 (키, 체중, 점수) | 이진 범주 (질병 유/무, 합격/불합격) |
| **예측값 범위** | -∞ ~ +∞ | 0 ~ 1 (확률) |
| **모델** | Y = β₀ + β₁X | logit(P) = β₀ + β₁X |
| **해석** | "X 1 증가 시 Y β₁ 증가" | "X 1 증가 시 Odds e^β₁배 증가" |
| **가정** | 선형성, 정규성(잔차), 등분산 | ❌ 정규성 불필요 |

**로지스틱 회귀 예시**:
```
Odds Ratio (OR) = 2.5, p < .001

해석: "흡연자는 비흡연자에 비해 질병 위험이 2.5배"
```

---

### 5.4 로지스틱 vs 다항 로지스틱

**로지스틱 회귀** (Binary):
```
종속변수: 합격 (1) vs 불합격 (0)

P(합격) = 1 / (1 + e^-(β₀ + β₁X))
```

**다항 로지스틱** (Multinomial):
```
종속변수: 직업 (A, B, C)

기준 범주 (Reference): C
모델 1: log(P(A) / P(C)) = β₀ + β₁X
모델 2: log(P(B) / P(C)) = γ₀ + γ₁X
```

---

### 5.5 회귀 vs 상관분석 (재강조)

**상관분석의 한계**:
```
X와 Y 상관 (r = .7, p < .001)

→ "X와 Y는 관련 있다" (O)
→ "X가 Y를 예측한다" (△ 조심)
→ "X가 Y를 유발한다" (✗ 인과관계 아님)
```

**회귀분석의 장점**:
- 예측 모델 제공
- 다변량 통제 (Multiple regression)
- 인과 관계 추론 가능 (실험 설계 시)

---

## 6. 카이제곱 검정 계열

### 6.1 두 가지 카이제곱

| 검정 타입 | 목적 | 데이터 구조 | 예시 | 검색어 |
|----------|------|-----------|------|--------|
| **독립성 검정** | 두 범주형 변수의 독립성 | 교차표 (2×2 이상) | 성별 × 투표 선택 | 'chi-square-independence' |
| **적합도 검정** | 관측 분포와 기대 분포 일치 | 1차원 빈도표 | 주사위 공정성 (1~6 균등?) | 'chi-square-goodness' |

---

### 6.2 카이제곱 vs Fisher's Exact Test

**카이제곱 검정**:
```
사용 조건:
- 기대빈도 ≥ 5 (모든 셀)
- n ≥ 20

장점: 빠른 계산
```

**Fisher's Exact Test**:
```
사용 조건:
- 2×2 표
- 작은 샘플 (n < 20) 또는 기대빈도 < 5

장점: 정확한 p-value (순열 기반)
```

**의사결정**:
```
2×2 표 확인
│
├─ 기대빈도 모두 ≥ 5 → 카이제곱 검정
└─ 기대빈도 일부 < 5 → Fisher's Exact Test ⭐
```

---

### 6.3 카이제곱 vs McNemar Test

| 특성 | 카이제곱 (독립성) | McNemar 검정 |
|------|---------------|-------------|
| **데이터 구조** | 독립 샘플 2×2 | ✅ 쌍 데이터 2×2 |
| **예시** | 성별(남/녀) × 찬성/반대 | 전(찬성/반대) × 후(찬성/반대) |
| **귀무가설** | "성별과 찬성 독립" | "전후 의견 변화 없음" |

**McNemar 사용 시기**:
- 동일 개체의 전후 비교 (Before-After)
- 쌍 매칭 데이터 (Matched Pairs)

**검색 키워드**: 'mcnemar'

## 7. 사후 검정 비교

### 7.1 사후 검정 종류 (ANOVA 유의 시)

| 사후 검정 | Type I Error 제어 | 검정력 | 사용 시기 | 검색어 |
|----------|------------------|--------|----------|--------|
| **Tukey HSD** | 균형 잡힌 | 중간 | 모든 쌍 비교 (가장 흔함) | 'tukey' |
| **Bonferroni** | 매우 보수적 | 낮음 | 소수의 계획된 비교 | 'bonferroni' |
| **Scheffé** | 가장 보수적 | 가장 낮음 | 복잡한 대조 (contrast) | 'scheffe' |
| **Dunnett** | 대조군 최적화 | 높음 (대조군 비교) | 여러 실험군 vs 1개 대조군 | 'dunnett' |
| **Games-Howell** | 균형 잡힌 | 중간 | 등분산 가정 위반 시 | 'games-howell' |
| **Dunn's test** | 비모수용 | 중간 | Kruskal-Wallis 후 | 'dunn' |

---

### 7.2 Tukey HSD vs Bonferroni

**Tukey HSD**:
```
- 모든 쌍 비교 (pairwise) 최적화
- 균등한 보수성
- ANOVA와 함께 가장 흔히 사용

예: 4개 그룹 → 6개 비교 (모두 검정)
```

**Bonferroni**:
```
- 조정된 α = α / k (k = 비교 횟수)
- 매우 보수적 (검정력 낮음)
- 계획된 소수 비교에 적합

예: 4개 그룹 중 3개만 비교 → α = .05/3 = .017
```

**선택 기준**:
- 모든 쌍 비교 → Tukey HSD ⭐
- 특정 비교만 (≤ 3개) → Bonferroni

---

### 7.3 Tukey vs Games-Howell

| 특성 | Tukey HSD | Games-Howell |
|------|----------|--------------|
| **등분산 가정** | ✅ 필요 | ❌ 불필요 |
| **샘플 크기** | 유사 시 최적 | 다를 때도 강건 |
| **Type I Error** | 제어 좋음 | 제어 좋음 |
| **사용 조건** | Levene p > .05 | Levene p ≤ .05 |

**의사결정**:
```
ANOVA 유의 (p < .05)
│
└─ Levene's Test
    ├─ p > .05 (등분산) → Tukey HSD
    └─ p ≤ .05 (등분산 위반) → Games-Howell ⭐
```

## 8. 종합 비교표

### 8.1 연구 질문별 방법 선택

| 연구 질문 | 정규성 충족 | 정규성 위반 | 카테고리 |
|----------|-----------|-----------|---------|
| 두 그룹 비교 (독립) | 독립표본 t-검정, Welch's t | Mann-Whitney U | 가설검정 |
| 두 그룹 비교 (쌍) | 대응표본 t-검정 | Wilcoxon Signed-Rank | 가설검정 |
| 세 그룹 이상 (독립) | 일원 ANOVA | Kruskal-Wallis | 분산분석 |
| 세 그룹 이상 (쌍) | 반복측정 ANOVA | Friedman | 분산분석 |
| 두 변수 상관 | Pearson r | Spearman ρ | 상관분석 |
| 예측 모델 (연속형 Y) | 선형 회귀 | - | 회귀분석 |
| 예측 모델 (범주형 Y) | 로지스틱 회귀 | - | 회귀분석 |
| 두 범주형 변수 독립성 | 카이제곱 검정 | - | 범주형 분석 |

---

### 8.2 주요 의사결정 포인트

```
1. 데이터 타입?
   ├─ 연속형 → 2단계
   └─ 범주형 → 카이제곱/Fisher

2. 정규성 검정
   ├─ 충족 → 모수 검정
   └─ 위반 → 비모수 검정

3. 독립성 확인
   ├─ 독립 샘플 → 독립표본 t/ANOVA
   └─ 쌍 데이터 → 대응표본 t/반복측정 ANOVA

4. 등분산성 (비교 연구)
   ├─ 충족 → 표준 t-test/ANOVA
   └─ 위반 → Welch 방법/Games-Howell

5. 사후 검정
   ├─ 모든 쌍 비교 → Tukey HSD
   └─ 계획된 비교 → Bonferroni
```
