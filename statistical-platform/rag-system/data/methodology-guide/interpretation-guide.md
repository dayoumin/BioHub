## 1. p-value 해석

### 1.1 정의

**p-value**: 귀무가설(H₀)이 참이라고 가정할 때, 관측된 결과(또는 더 극단적인 결과)를 얻을 확률

```
p-value = P(관측 데이터 또는 더 극단적인 데이터 | H₀가 참)
```

---

### 1.2 올바른 해석

#### ✅ 맞는 해석

```
예: t-test 결과 p = 0.03

"귀무가설(두 그룹 평균이 같음)이 참이라면,
현재 관측된 차이(또는 더 큰 차이)를 얻을 확률은 3%이다."
```

**의사결정**:
- p < 0.05 → **귀무가설 기각** (H₀ 기각)
- p ≥ 0.05 → **귀무가설 기각 실패** (H₀ 채택 아님!)

---

#### ❌ 흔한 오해

**오해 1**: "p = 0.03이므로 귀무가설이 참일 확률이 3%이다"
- **❌ 틀림**: p-value는 H₀가 **이미 참이라고 가정**한 조건부 확률
- **✅ 올바른 이해**: H₀ 아래에서 이 데이터를 볼 확률이 3%

**오해 2**: "p = 0.03이므로 대립가설이 참일 확률이 97%이다"
- **❌ 틀림**: p-value는 H₁의 확률을 말하지 않음
- **✅ 올바른 이해**: 베이지안 통계에서는 사후확률(Posterior Probability)로 계산 가능

**오해 3**: "p = 0.049는 유의하고, p = 0.051은 유의하지 않으므로 큰 차이"
- **❌ 틀림**: 0.05는 임의의 기준 (arbitrary threshold)
- **✅ 올바른 이해**: p-value는 연속적 증거의 강도

---

### 1.3 유의수준 (Significance Level, α)

**α = 0.05의 의미**:
```
귀무가설이 참일 때, 잘못 기각할 확률(Type I Error)을
5% 이하로 제한
```

#### 유의수준 선택 가이드

| α 수준 | 사용 시기 | Type I Error | 보수성 |
|--------|----------|--------------|--------|
| **0.05** | 일반 연구 (가장 흔함) | 5% | 중간 |
| **0.01** | 엄격한 기준 필요 시 | 1% | 보수적 |
| **0.10** | 탐색적 연구 (Exploratory) | 10% | 관대 |
| **0.001** | 대규모 데이터 (Bonferroni 보정 후) | 0.1% | 매우 보수적 |

---

### 1.4 p-value 보고 지침

#### 권장 형식

**방법 1: 정확한 p-value 보고 (권장)**
```
t(28) = 2.45, p = .021
```

**방법 2: 부등호 사용 (p < .001)**
```
t(28) = 4.12, p < .001
```

#### ❌ 피해야 할 형식

```
p = .000        (❌ 정확히 0은 불가능, p < .001 사용)
p = NS          (❌ 정확한 p-value 보고)
p < .05*        (❌ 별표 대신 정확한 값)
```

---

### 1.5 의사결정 플로우차트

```
시작: p-value 계산 결과
│
├─ p < α (예: p = 0.03 < 0.05)
│   ├─ 결론: 귀무가설 기각
│   ├─ 해석: "통계적으로 유의미한 차이가 있다"
│   └─ 다음 단계: 효과 크기 확인 →
│
└─ p ≥ α (예: p = 0.12 ≥ 0.05)
    ├─ 결론: 귀무가설 기각 실패
    ├─ 해석: "통계적으로 유의미한 차이를 발견하지 못했다"
    │         (≠ "차이가 없다" 증명)
    └─ 다음 단계: 검정력(Power) 확인
```

## 2. 효과 크기 (Effect Size)

### 2.1 정의

**효과 크기**: 집단 간 차이 또는 관계의 **실질적인 크기**(Practical Significance)

```
p-value = 통계적 유의성 (샘플 크기에 민감)
Effect Size = 실질적 중요성 (샘플 크기 독립적)
```

---

### 2.2 주요 효과 크기 지표

#### 2.2.1 Cohen's d (t-test용)

**정의**: 두 그룹 평균 차이를 표준편차로 나눈 값

```
d = (Mean₁ - Mean₂) / Pooled SD

해석:
d = 0.2  → 작은 효과 (Small)
d = 0.5  → 중간 효과 (Medium)
d = 0.8  → 큰 효과 (Large)
```

**예시**:
```
그룹 A 평균 = 75, SD = 10
그룹 B 평균 = 80, SD = 10
d = (80 - 75) / 10 = 0.5 (중간 효과)

→ "두 그룹은 0.5 표준편차만큼 차이"
```

---

#### 2.2.2 η² (Eta-squared, ANOVA용)

**정의**: 총 변동 중 그룹 간 차이로 설명되는 비율

```
η² = SS_between / SS_total

해석:
η² = 0.01  → 작은 효과 (1% 설명)
η² = 0.06  → 중간 효과 (6% 설명)
η² = 0.14  → 큰 효과 (14% 설명)
```

**예시**:
```
ANOVA 결과: F(2, 57) = 5.23, p = .008, η² = .155

→ "그룹 변수는 결과 변동의 15.5%를 설명 (큰 효과)"
```

---

#### 2.2.3 Partial η² (부분 에타제곱)

**이원 ANOVA, ANCOVA에서 사용**

```
Partial η² = SS_effect / (SS_effect + SS_error)

→ 다른 요인의 영향을 제거한 순수 효과
```

---

#### 2.2.4 r (Pearson 상관계수)

**정의**: 두 변수 간 선형 관계의 강도

```
r 범위: -1 ~ +1

해석:
|r| = 0.10  → 약한 상관 (Weak)
|r| = 0.30  → 중간 상관 (Medium)
|r| = 0.50  → 강한 상관 (Strong)
```

**예시**:
```
r = .42, p = .003

→ "중간~강한 양의 상관 (r² = .176, 17.6% 공유 분산)"
```

---

#### 2.2.5 Cramér's V (카이제곱용)

**정의**: 범주형 변수 간 연관성의 강도

```
V = √(χ² / (n × df*))
df* = min(rows-1, cols-1)

해석 (df* = 1 기준):
V = 0.10  → 작은 효과
V = 0.30  → 중간 효과
V = 0.50  → 큰 효과
```

---

### 2.3 효과 크기 비교표

| 통계 방법 | 효과 크기 | Small | Medium | Large |
|----------|----------|-------|--------|-------|
| t-test | Cohen's d | 0.2 | 0.5 | 0.8 |
| ANOVA | η² | 0.01 | 0.06 | 0.14 |
| 상관분석 | r | 0.10 | 0.30 | 0.50 |
| 카이제곱 (df=1) | Cramér's V | 0.10 | 0.30 | 0.50 |
| 회귀분석 | R² | 0.02 | 0.13 | 0.26 |

**출처**: Cohen, J. (1988)

---

### 2.4 효과 크기 보고 예시

```
독립표본 t-검정:
"실험군(M = 82.5, SD = 12.3)이 대조군(M = 75.2, SD = 11.8)보다
유의미하게 높은 점수를 보였다, t(58) = 2.45, p = .017, d = 0.61 (중간 효과)."

일원 ANOVA:
"세 그룹 간 유의미한 차이가 있었다, F(2, 87) = 8.23, p < .001, η² = .159 (큰 효과)."

Pearson 상관:
"학습 시간과 성적 간 유의미한 양의 상관이 있었다, r(48) = .42, p = .003 (중간 상관)."
```

## 3. 신뢰구간 (Confidence Interval)

### 3.1 정의

**95% 신뢰구간**: 동일한 실험을 100번 반복할 때, 95번은 참값(모수)이 이 구간에 포함

```
예: 평균 차이 = 5.2, 95% CI [1.3, 9.1]

해석:
"실제 평균 차이는 95%의 신뢰도로 1.3~9.1 사이에 있다."
```

---

### 3.2 신뢰구간과 p-value의 관계

```
95% CI와 p-value (α = .05)의 관계:

├─ CI가 0을 포함 안 함 → p < .05 (유의)
└─ CI가 0을 포함 → p ≥ .05 (비유의)

예시:
1. 평균 차이 = 5.2, 95% CI [1.3, 9.1] → p < .05 ✓
2. 평균 차이 = 2.1, 95% CI [-0.5, 4.7] → p ≥ .05 ✗
   (0이 구간에 포함)
```

---

### 3.3 신뢰구간의 장점

**1. 효과 크기의 불확실성 표현**
```
평균 차이 = 5.2, 95% CI [1.3, 9.1]

→ "차이가 1.3만큼 작을 수도, 9.1만큼 클 수도 있다"
→ p-value만으로는 알 수 없는 정보
```

**2. 임상적/실질적 유의성 판단**
```
최소 임상적 유의 차이(MCID) = 3점

케이스 A: 평균 차이 = 5.2, 95% CI [4.1, 6.3]
→ 전체 구간이 MCID > 3 → 임상적으로 유의 ✓

케이스 B: 평균 차이 = 3.8, 95% CI [0.5, 7.1]
→ 구간이 MCID 포함 → 임상적 유의성 불확실 ⚠️
```

---

### 3.4 신뢰구간 보고 예시

```
t-test:
"실험군이 대조군보다 평균 7.3점 높았다
(95% CI [2.1, 12.5]), t(58) = 2.85, p = .006."

ANOVA (Tukey 사후 검정):
"그룹 A가 그룹 B보다 평균 12.4점 높았다
(95% CI [5.2, 19.6]), p = .001."

상관분석:
"r = .42 (95% CI [.15, .63]), p = .003."
```

## 4. 검정력 (Statistical Power)

### 4.1 정의

**검정력 (1-β)**: 실제 효과가 있을 때, 그것을 올바르게 발견할 확률

```
Power = P(H₀ 기각 | H₁이 참)

표준 권장: Power ≥ 0.80 (80%)
```

---

### 4.2 Type I/II 에러와의 관계

|  | H₀가 참 | H₁이 참 |
|--|---------|---------|
| **H₀ 기각** | ❌ Type I Error (α) | ✅ 올바른 기각 (1-β, Power) |
| **H₀ 기각 실패** | ✅ 올바른 채택 (1-α) | ❌ Type II Error (β) |

---

### 4.3 검정력에 영향을 주는 요인

```
Power ↑ 영향 요인:
1. 샘플 크기 (n) ↑
2. 효과 크기 (d) ↑
3. 유의수준 (α) ↑ (Type I 에러 증가 단점)
4. 측정 오차 (measurement error) ↓
```

---

### 4.4 비유의 결과 (p ≥ .05) 해석

#### Case A: 높은 검정력 (Power = .80)
```
p = .12, Power = .85, n = 120

→ "충분한 샘플로 검증했으나 차이를 발견하지 못했다"
→ "실제로 차이가 없을 가능성 높음"
```

#### Case B: 낮은 검정력 (Power = .30)
```
p = .12, Power = .35, n = 25

→ "샘플이 작아서 차이를 발견하지 못했을 수 있다"
→ "Type II Error 가능성 높음 (실제 차이 있으나 못 찾음)"
→ "샘플 크기 증가 필요"
```

---

### 4.5 검정력 분석 사용 시기

#### 사전 검정력 분석 (A Priori)
```
질문: "유의미한 효과를 발견하려면 몇 명이 필요한가?"

입력:
- Effect Size: d = 0.5 (예상 효과)
- α = .05
- Power = .80

출력: 필요 샘플 크기 (예: n = 64 per group)
```

**검색 키워드**: 'power-analysis'
**카테고리**: 검정력 분석

#### 사후 검정력 분석 (Post Hoc)
⚠️ **주의**: 일부 통계학자는 권장하지 않음
- 대안: 신뢰구간 + 효과 크기 보고

## 5. 통계적 유의성 vs 실질적 유의성

### 5.1 차이점

| 기준 | 통계적 유의성 | 실질적 유의성 |
|------|--------------|--------------|
| **정의** | p < .05 (샘플에서 우연으로 나타날 확률 낮음) | 효과 크기가 충분히 큼 (실용적 가치) |
| **지표** | p-value | Effect Size, CI, MCID |
| **샘플 크기 영향** | ✅ 민감 (n↑ → p↓) | ❌ 독립적 |
| **의사결정** | 귀무가설 기각 여부 | 개입/정책 실시 여부 |

---

### 5.2 불일치 케이스

#### Case A: 통계적 유의 but 실질적 무의미
```
n = 10,000 (대규모)
평균 차이 = 0.5점 (100점 만점)
p < .001 (매우 유의)
d = 0.05 (작은 효과)

→ 통계적으로는 유의하나, 실질적 차이는 미미
→ "0.5점 차이는 실용적으로 무시 가능"
```

#### Case B: 통계적 비유의 but 실질적 유의
```
n = 15 (소규모)
평균 차이 = 12점 (100점 만점)
p = .08 (비유의)
d = 0.75 (큰 효과)

→ 통계적으로는 비유의하나, 효과 크기는 큼
→ "샘플 크기 부족으로 유의성 달성 실패"
→ "추가 연구 필요 (검정력 부족)"
```

---

### 5.3 종합 의사결정 트리

```
Step 1: p-value 확인
├─ p < .05 → Step 2
└─ p ≥ .05 → 검정력 확인 → 샘플 크기 충분?

Step 2: 효과 크기 확인
├─ 큰 효과 (d > 0.8 또는 η² > 0.14) → 실질적 유의 ✓
├─ 중간 효과 (d = 0.5~0.8) → 맥락 고려
└─ 작은 효과 (d < 0.5) → 실용성 의심

Step 3: 신뢰구간 확인
├─ CI가 실질적 유의 기준(MCID) 벗어남 → 유의 ✓
└─ CI가 MCID 포함 → 불확실

최종 결정: p-value + Effect Size + CI 종합 판단
```

## 6. 일반적인 오해와 주의사항

### 6.1 p-value 관련 오해

#### 오해 1: "p = .05는 마법의 숫자"
```
❌ 잘못된 생각:
"p = .049는 유의, p = .051은 완전히 다름"

✅ 올바른 이해:
"p-value는 연속적 증거의 강도, .05는 임의의 경계선"
```

---

#### 오해 2: "비유의 = 차이 없음 증명"
```
❌ 잘못된 해석:
"p = .12이므로 두 그룹은 차이가 없다"

✅ 올바른 해석:
"p = .12이므로 차이가 있다는 충분한 증거를 찾지 못했다"
(부재의 증거 ≠ 증거의 부재)
```

---

#### 오해 3: "p-value가 작을수록 효과가 크다"
```
예시:
연구 A: p < .001, d = 0.15 (n = 5000)
연구 B: p = .04, d = 0.65 (n = 40)

→ 연구 A가 더 유의하지만, 효과는 연구 B가 훨씬 큼
→ p-value ≠ 효과 크기
```

---

### 6.2 다중 비교 문제

**문제**: 여러 번 검정하면 Type I Error 증가

```
1개 검정: P(Type I Error) = .05
10개 검정: P(최소 1개 Type I Error) ≈ .40 (40%)

→ "유의한 결과를 찾을 때까지 계속 검정" (p-hacking)
```

#### 해결 방법

**Bonferroni 보정**:
```
조정된 α = α / k (k = 비교 횟수)

예: 10개 비교 → α = .05 / 10 = .005
→ p < .005일 때만 유의
```

**검색 키워드**: 'bonferroni', 'multiple-comparisons'

---

### 6.3 보고 체크리스트

**필수 보고 항목**:
- [ ] p-value (정확한 값 또는 p < .001)
- [ ] 효과 크기 (Cohen's d, η², r 등)
- [ ] 신뢰구간 (95% CI)
- [ ] 샘플 크기 (n)
- [ ] 검정 통계량 (t, F, χ² 등)
- [ ] 자유도 (df)

**예시 (APA 형식)**:
```
"실험군(M = 82.5, SD = 12.3)이 대조군(M = 75.2, SD = 11.8)보다
```

## 7. Quick Reference

### 7.1 해석 순서

```
1. p-value 확인 → 통계적 유의성
2. 효과 크기 확인 → 실질적 중요성
3. 신뢰구간 확인 → 불확실성 범위
4. 검정력 확인 (비유의 시) → Type II Error 가능성
5. 종합 판단 → 최종 결론
```

### 7.2 의사결정 매트릭스

| p-value | Effect Size | 해석 | 조치 |
|---------|-------------|------|------|
| p < .05 | 큰 효과 | ⭐ 강한 증거 | 결과 적용 권장 |
| p < .05 | 중간 효과 | ✓ 유의미 | 맥락 고려 후 적용 |
| p < .05 | 작은 효과 | ⚠️ 주의 | 실용성 검토 (샘플 크기?) |
| p ≥ .05 | 큰 효과 | ⚠️ 검정력 부족? | 샘플 증가 후 재검증 |
| p ≥ .05 | 작은 효과 | ✗ 증거 부족 | 차이 없을 가능성 |
