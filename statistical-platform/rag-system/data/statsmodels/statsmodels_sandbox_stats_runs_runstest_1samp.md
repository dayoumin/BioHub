use runs test on binary discretized data above/below cutoff 

Parameters:[¶](https://www.statsmodels.org/stable/generated/statsmodels.sandbox.stats.runs.runstest_1samp.html#statsmodels.sandbox.stats.runs.runstest_1samp-parameters "Permalink to this headline") 
     

**x**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
data, numeric 

**cutoff**{‘mean’, ‘median’} `or` `number` 
    
This specifies the cutoff to split the data into large and small values. 

**correction**[ bool](https://docs.python.org/3/library/stdtypes.html#bltin-boolean-values "\(in Python v3.12\)") 
    
Following the SAS manual, for samplesize below 50, the test statistic is corrected by 0.5. This can be turned off with correction=False, and was included to match R, tseries, which does not use any correction.

**p-value**[`float`](https://docs.python.org/3/library/functions.html#float "\(in Python v3.12\)")
