**endog**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
endog is an 1-d vector of the endogenous response. endog can contain strings, ints, or floats or may be a pandas Categorical Series. Note that if it contains strings, every distinct string will be a category. No stripping of whitespace is done. 

**exog**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
A nobs x k array where nobs is the number of observations and k is the number of regressors. An intercept is not included by default and should be added by the user. See statsmodels.tools.add_constant. 

**missing**[`str`](https://docs.python.org/3/library/stdtypes.html#str "\(in Python v3.12\)") 
    
Available options are ‘none’, ‘drop’, and ‘raise’. If ‘none’, no nan checking is done. If ‘drop’, any observations with nans are dropped. If ‘raise’, an error is raised. Default is ‘none’. 

**check_rank**[ bool](https://docs.python.org/3/library/stdtypes.html#bltin-boolean-values "\(in Python v3.12\)") 
    
Check exog rank to determine model degrees of freedom. Default is True. Setting to False reduces model initialization time when exog.shape[1] is large. 

Attributes:[¶](https://www.statsmodels.org/stable/generated/statsmodels.discrete.discrete_model.MNLogit.html#statsmodels.discrete.discrete_model.MNLogit-attributes "Permalink to this headline") 
     

**endog**[`ndarray`](https://numpy.org/doc/stable/reference/generated/numpy.ndarray.html#numpy.ndarray "\(in NumPy v2.1\)") 
    
A reference to the endogenous response variable 

**exog**[`ndarray`](https://numpy.org/doc/stable/reference/generated/numpy.ndarray.html#numpy.ndarray "\(in NumPy v2.1\)") 
    
A reference to the exogenous design. 

**J**[`float`](https://docs.python.org/3/library/functions.html#float "\(in Python v3.12\)") 
    
The number of choices for the endogenous variable. Note that this is zero-indexed. 

**K**[`float`](https://docs.python.org/3/library/functions.html#float "\(in Python v3.12\)") 
    
The actual number of parameters for the exogenous design. Includes the constant if the design has one. 

**names**[`dict`](https://docs.python.org/3/library/stdtypes.html#dict "\(in Python v3.12\)") 
    
A dictionary mapping the column number in wendog to the variables in endog. 

**wendog**[`ndarray`](https://numpy.org/doc/stable/reference/generated/numpy.ndarray.html#numpy.ndarray "\(in NumPy v2.1\)") 
    
An n x j array where j is the number of unique categories in endog. Each column of j is a dummy variable indicating the category of each observation. See names for a dictionary mapping each column to its category.
