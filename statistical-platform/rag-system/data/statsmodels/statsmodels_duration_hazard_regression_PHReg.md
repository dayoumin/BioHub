Cox Proportional Hazards Regression Model
The Cox PH Model is for right censored data. 

Parameters:[¶](https://www.statsmodels.org/stable/generated/statsmodels.duration.hazard_regression.PHReg.html#statsmodels.duration.hazard_regression.PHReg-parameters "Permalink to this headline") 
     

**endog**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
The observed times (event or censoring) 

**exog** 2D [array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
The covariates or exogeneous variables 

**status**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
The censoring status values; status=1 indicates that an event occurred (e.g. failure or death), status=0 indicates that the observation was right censored. If None, defaults to status=1 for all cases. 

**entry**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
The entry times, if left truncation occurs 

**strata**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
Stratum labels. If None, all observations are taken to be in a single stratum. 

**ties**[`str`](https://docs.python.org/3/library/stdtypes.html#str "\(in Python v3.12\)") 
    
The method used to handle tied times, must be either ‘breslow’ or ‘efron’.

Array of offset values

The method used to handle missing data

Notes
Proportional hazards regression models should not include an explicit or implicit intercept. The effect of an intercept is not identified using the partial likelihood approach.
endog, event, strata, entry, and the first dimension of exog all must have the same length
