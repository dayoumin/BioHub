# statsmodels.tools.tools.add_constant[¶](https://www.statsmodels.org/stable/generated/statsmodels.tools.tools.add_constant.html#statsmodels-tools-tools-add-constant "Link to this heading") 

statsmodels.tools.tools.add_constant(_[data](https://www.statsmodels.org/stable/generated/statsmodels.tools.tools.add_constant.html#statsmodels.tools.tools.add_constant "statsmodels.tools.tools.add_constant.data \(Python parameter\)")_ , _[prepend](https://www.statsmodels.org/stable/generated/statsmodels.tools.tools.add_constant.html#statsmodels.tools.tools.add_constant "statsmodels.tools.tools.add_constant.prepend \(Python parameter\)") =`True`_, _[has_constant](https://www.statsmodels.org/stable/generated/statsmodels.tools.tools.add_constant.html#statsmodels.tools.tools.add_constant "statsmodels.tools.tools.add_constant.has_constant \(Python parameter\)") =`'skip'`_)[[source]](https://www.statsmodels.org/stable/_modules/statsmodels/tools/tools.html#add_constant)[¶](https://www.statsmodels.org/stable/generated/statsmodels.tools.tools.add_constant.html#statsmodels.tools.tools.add_constant "Link to this definition") 
    
Add a column of ones to an array. 

Parameters:[¶](https://www.statsmodels.org/stable/generated/statsmodels.tools.tools.add_constant.html#statsmodels.tools.tools.add_constant-parameters "Permalink to this headline") 
     

**data**[ array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)") 
    
A column-ordered design matrix. 

**prepend**[ bool](https://docs.python.org/3/library/stdtypes.html#bltin-boolean-values "\(in Python v3.12\)") 
    
If true, the constant is in the first column. Else the constant is appended (last column). 

**has_constant**[`str`](https://docs.python.org/3/library/stdtypes.html#str "\(in Python v3.12\)") {‘raise’, ‘add’, ‘skip’} 
    
Behavior if `data` already has a constant. The default will return data without adding another constant. If ‘raise’, will raise an error if any column has a constant value. Using ‘add’ will add a column of 1s if a constant column is present. 

Returns:[¶](https://www.statsmodels.org/stable/generated/statsmodels.tools.tools.add_constant.html#statsmodels.tools.tools.add_constant-returns "Permalink to this headline") 
     

[array_like](https://numpy.org/doc/stable/glossary.html#term-array_like "\(in NumPy v2.1\)")
    
The original values with a constant (column of ones) as the first or last column. Returned value type depends on input type.
Notes
When the input is a pandas Series or DataFrame, the added column’s name is ‘const’.
