RAG DB Generation Flow

Statistical Platform - RAG System Database Build Process

1
Document Collection
Manual

Crawl documentation from SciPy, statsmodels, NumPy, OpenIntro, and Pingouin.

# Navigate to scripts directory cd rag-system/scripts # Run crawlers python crawl_scipy_batch.py python crawl_statsmodels_batch.py python crawl_numpy_batch.py python crawl_openintro_batch.py python crawl_pingouin_batch.py
Output
rag-system/data/{scipy,statsmodels,numpy,openintro,pingouin}/*.md
2
SQLite DB Build
Manual

Parse markdown files and create SQLite database with documents table.

cd rag-system/scripts python build_sqlite_db.py
Output
rag-system/data/rag.db (111 documents, ~5.4 MB)
3
Embedding Generation
Manual

Generate vector embeddings using Ollama models (qwen3-embedding, mxbai-embed-large).

cd rag-system/scripts python generate_embeddings.py
Output
rag-system/data/vector-qwen3-embedding-0.6b.db (~5.4 MB)
rag-system/data/vector-mxbai-embed-large.db
4
Build Time (Automatic)
Auto

Prebuild hooks automatically copy DB files and generate metadata JSON.

# This runs automatically during npm run build npm run build # Prebuild executes: # 1. copy-db.js: rag-system/data/*.db ‚Üí public/rag-data/ # 2. generate-metadata.js: creates vector-stores.json
Output
public/rag-data/rag.db
public/rag-data/vector-*.db
public/rag-data/vector-stores.json
