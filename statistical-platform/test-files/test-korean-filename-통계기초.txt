# 통계학 기초

## 기술통계학

기술통계학은 데이터를 요약하고 설명하는 통계학의 한 분야입니다.

### 중심경향성 측정치

1. **평균 (Mean)**
   - 모든 값의 합을 개수로 나눈 값
   - 공식: μ = Σx / n

2. **중앙값 (Median)**
   - 데이터를 크기 순으로 정렬했을 때 중간 값

3. **최빈값 (Mode)**
   - 가장 자주 나타나는 값

### 산포도 측정치

1. **분산 (Variance)**
   - 평균으로부터 떨어진 정도의 평균
   - 공식: σ² = Σ(x - μ)² / n

2. **표준편차 (Standard Deviation)**
   - 분산의 제곱근
   - 공식: σ = √(σ²)

3. **범위 (Range)**
   - 최댓값 - 최솟값

## 추론통계학

추론통계학은 표본으로부터 모집단의 특성을 추정하는 통계학의 한 분야입니다.

### 가설검정

1. 귀무가설 (H₀) 설정
2. 대립가설 (H₁) 설정
3. 유의수준 (α) 결정 (보통 0.05)
4. 검정통계량 계산
5. p-값 계산
6. 결론 도출

### 주요 검정 방법

- t-검정: 두 집단의 평균 비교
- ANOVA: 세 개 이상 집단의 평균 비교
- 카이제곱 검정: 범주형 변수의 독립성 검정
- 상관분석: 두 변수 간의 관계 측정
