포아송 회귀분석이란?

종속변수가 비음의 정수인 카운트 데이터(count data)를 분석하는 일반화선형모델(GLM)입니다. 포아송 분포를 가정하며 로그 연결함수를 사용하여 발생률(rate)을 모델링합니다.

주요 특징

  • • 카운트 데이터 전용
  • • 포아송 분포 가정
  • • 로그 연결함수 사용
  • • 발생률 비(Rate Ratio) 해석

적용 예시

  • • 질병 발생 건수
  • • 교통사고 발생 횟수
  • • 고객 방문 횟수
  • • 결함 발생 개수

일반 선형회귀 vs 포아송 회귀

일반 선형회귀

  • • 연속형 종속변수
  • • 정규분포 가정
  • • 항등 연결함수
  • • 음수 예측값 가능

포아송 회귀

  • • 카운트 종속변수
  • • 포아송분포 가정
  • • 로그 연결함수
  • • 항상 양수 예측값

포아송 회귀의 가정

1포아송 분포: 평균과 분산이 같음
2독립성: 관측치들이 서로 독립
3선형성: 로그(평균)이 예측변수와 선형관계
4과산포 없음: 분산 = 평균 (과산포 시 음이항 회귀 고려)

카운트 데이터 예시

의료

병원 방문 횟수, 발작 횟수, 합병증 건수

품질관리

결함 개수, 불량품 수, 클레임 건수

마케팅

구매 횟수, 클릭 수, 문의 건수

포아송 회귀에서 과산포(Overdispersion)가 발생하면 표준오차가 과소추정될 수 있습니다. 이 경우 준-포아송 모델이나 음이항 회귀분석을 고려해야 합니다.
