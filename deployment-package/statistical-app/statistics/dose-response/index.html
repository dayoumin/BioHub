용량-반응 분석
용량과 생물학적 반응 간의 관계를 수학적 모델로 분석하여 EC50, IC50 등 핵심 매개변수를 추정합니다
방법론 이해

용량-반응 분석은 약물, 독소, 또는 기타 화학물질의 용량과 생물학적 반응 간의 관계를 정량적으로 분석하는 방법입니다.

언제 사용하나요?
  • • 약물의 효력 및 독성 평가
  • • EC50, IC50, ED50 값 결정
  • • 용량-반응 곡선 모델링
  • • 생물학적 활성 비교 연구
주요 특징
  • • S자형(시그모이드) 곡선 분석
  • • 다양한 수학적 모델 지원
  • • 신뢰구간 및 통계 검정
  • • 매개변수 제약조건 설정 가능
모델 선택

데이터의 특성과 연구 목적에 따라 적절한 모델을 선택하세요.

고전적인 S자형 용량-반응 곡선 (4PL)

y = D + (A-D)/(1+(x/C)^B)

매개변수: A (최대값), B (힐 기울기), C (EC50), D (최소값)

적용: 대부분의 용량-반응 실험

최소값이 0으로 고정된 로지스틱 모델 (3PL)

y = A/(1+(x/C)^B)

매개변수: A (최대값), B (힐 기울기), C (EC50)

적용: 최소값이 0인 억제/활성화 실험

비대칭 S자형 곡선 모델

y = D + (A-D) * exp(-exp(B*(log(x)-log(C))))

매개변수: A (최대값), B (기울기), C (변곡점), D (최소값)

적용: 독성학, 생존 분석

성장 곡선 모델

y = A * exp(-exp(B*(C-x)))

매개변수: A (최대값), B (성장률), C (변곡점)

적용: 세포 성장, 종양 성장

이중 용량-반응 곡선

y = D + (A1-D)/(1+(x/C1)^B1) + (A2-D)/(1+(x/C2)^B2)

매개변수: A1, A2 (최대값들), B1, B2 (기울기들), C1, C2 (EC50들), D (최소값)

적용: 복합 수용체, 다중 작용점
