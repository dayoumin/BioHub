NIFS 통계 분석 플랫폼
공분산분석
Analysis of Covariance (ANCOVA)
단계 1/4분석 방법
공분산분석 소개
공변량을 통제한 집단 간 비교 분석
분석 목적

공변량(covariate)의 영향을 통제한 후 집단 간 평균 차이를 검정합니다.

  • • 사전 점수나 연령 등의 영향 제거
  • • 순수한 처치 효과 확인
  • • 검정력 향상 효과
  • • 수정된 평균(adjusted means) 비교
적용 예시

교육 효과

사전 점수를 통제한 학습법 비교

임상 시험

기저선 값을 통제한 치료 효과

언제 사용하나요?
• 집단 간 비교에서 영향을 미칠 수 있는 변수가 있을 때
• 사전-사후 설계에서 사전 점수를 통제하고 싶을 때
• ANOVA의 검정력을 향상시키고 싶을 때
• 무작위 배정이 완전하지 않은 준실험 설계

주요 가정

  • • 선형성: 공변량과 종속변수 간 선형관계
  • • 회귀직선 동질성: 집단별 회귀계수가 동일
  • • 정규성: 잔차가 정규분포
  • • 등분산성: 집단 간 잔차 분산 동일
  • • 독립성: 공변량과 처치가 독립적
