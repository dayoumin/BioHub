NIFS 통계 분석 플랫폼

도움말 센터

통계 분석 플랫폼 사용법과 자주 묻는 질문들을 확인하세요.

통계 분석 플랫폼 사용법
5단계로 완료되는 체계적인 분석 과정
1
데이터 파일 업로드 (CSV, Excel 지원)
2
데이터 품질 확인 및 전처리
3
적절한 통계 분석 방법 선택
4
SciPy 기반 정확한 분석 실행
5
상세한 결과 해석 및 다음 단계 안내
데이터 준비 가이드
올바른 데이터 형식으로 업로드하는 방법
중요: 데이터의 첫 번째 행에는 반드시 열 이름(변수명)을 입력해주세요. 예: "나이", "성별", "점수" 등

✅ 올바른 예시

나이,성별,점수
25,남성,85
30,여성,92
28,남성,78

❌ 잘못된 예시

25,남성,85점
30세,여성,92
28,Female,78
평균,28.5,85
