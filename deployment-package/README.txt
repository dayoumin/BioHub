🚀 빠른 시작 (3가지 방법)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[방법 1] 가장 간단함 (권장)
─────────────────────────
1. start.bat (Windows) 또는 start.sh (macOS/Linux) 더블클릭
2. 자동으로 앱이 열립니다
3. 웹 브라우저에서 http://localhost:8000 접속

[방법 2] 직접 파일 열기
──────────────────────
1. statistical-app 폴더 열기
2. index.html 을 웹 브라우저로 열기
3. (Chrome/Edge 권장)

[방법 3] 수동 설정 (고급 사용자)
───────────────────────────────
1. 터미널 열기
2. 이 폴더로 이동
3. 명령어 실행:
   Windows: start.bat
   macOS/Linux: ./start.sh

또는:
   cd statistical-app
   python -m http.server 8000
   브라우저에서 http://localhost:8000 접속

🎯 주요 기능
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ 기술 통계량
  - 평균, 표준편차, 왜도, 첨도 등

✓ 가설 검정
  - t-test, ANOVA, 카이제곱검정 등

✓ 회귀 분석
  - 선형 회귀, 다중 회귀, 로지스틱 회귀

✓ 비모수 검정
  - Mann-Whitney, Kruskal-Wallis 등

✓ 고급 분석
  - 상관분석, 주성분분석, 클러스터링

✓ AI 어시스턴트 (Ollama 설치 시)
  - 분석 결과 해석
  - 보고서 생성
  - 통계 용어 설명

✓ 시각화
  - 차트, 그래프, 테이블 자동 생성


📊 사용 순서
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 앱 실행
   → start.bat 또는 index.html 열기

2. 데이터 로드
   → CSV 파일 업로드 또는 샘플 데이터 사용

3. 분석 선택
   → 왼쪽 메뉴에서 분석 유형 선택

4. 변수 설정
   → 필요한 변수 선택

5. 분석 실행
   → "Analyze" 버튼 클릭

6. 결과 확인
   → 표와 차트 보기

7. 보고서 생성 (선택)
   → AI 어시스턴트로 해석 요청
