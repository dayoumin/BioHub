## 📋 프로세스 작성 후 (노드 배치)

```
┌──────────────────────────────────────────────────────────────────────────────┐
```

```
│                        │

│     30%      70%      │
└────┬───────────┬───────┘
     │           │
     ▼           ▼
 [Step 1]    [Step 3]
 재작성      계약체결
```

**노드 데이터**:
```json
{
  "id": "step-2",
  "type": "branch",
  "title": "법무 검토",
  "branch": {
    "question": "수정이 필요한가요?",
    "yes": "step-1",
    "no": "step-3"
  }
}
```

## 🎨 노드 스타일 가이드

### 일반 작업 노드
```
┌──────────────────────┐
│ 📋 입찰 공고 작성     │  ← 파란색 테두리
│                      │
│ 입찰 공고문을 작성   │  ← 회색 설명
│ 하고 예산을 확인     │
│                      │
│ ✓ 3개 체크리스트     │  ← 메타 정보
│ 📎 2개 서류          │
└──────────────────────┘
```

### 조건 분기 노드
```
┌──────────────────────┐
│ 🔀 법무 검토          │  ← 노란색 테두리
│                      │
│ ❓ 수정 필요한가요?   │  ← 빨간색 질문
│                      │
│   Yes ↶ │ No ↓     │  ← 양쪽 화살표
└──────────────────────┘
```

### 파일 첨부 노드
```
┌──────────────────────┐
│ 📎 서류 제출          │  ← 녹색 테두리

│ ✅ 예산 증빍 (XLSX)   │
│ ☐ 사업계획서 (DOCX)  │
```

## 🎯 사용자 인터랙션

### 드래그 시작
```
팔레트에서 "일반 작업" 클릭 + 드래그

마우스 커서: 🖱️ → ✊

캔버스에 반투명 미리보기:
┌───────────────┐
│ 📋 새 작업     │  ← 50% 투명도
│               │
│ (드롭하여 추가)│
└───────────────┘
```

### 드롭
```
캔버스에 드롭 → 노드 생성 + 속성 편집기 자동 열림

┌──────────────────────┐     ┌─────────────────┐
│ 캔버스               │     │ 속성 편집기      │
│                      │     │                 │
│ ┌──────────────────┐ │     │ 제목: _______   │
│ │ 📋 새 작업        │ │ →  │ 설명: _______   │
│ │                  │ │     │ 체크리스트:     │
│ │ (제목 입력 필요) │ │     │ [+ 항목 추가]   │
│ └──────────────────┘ │     │                 │
│                      │     │ [저장]          │
└──────────────────────┘     └─────────────────┘
```

### 노드 연결
```
노드 하단 핸들 클릭 + 다른 노드로 드래그

┌──────────┐
│ Step 1   │
└─────●────┘  ← 핸들 클릭
      │
      ▼ (드래그 중 선 표시)
      │
┌─────●────┐
│ Step 2   │
└──────────┘

연결 완료 → 화살표 표시
```
