# 스마트분석 페이지 - 통합 테스트 체크리스트

## 🎯 목적
실제 브라우저에서 에러 처리 및 Tooltip 기능이 정상 작동하는지 검증

## 📋 테스트 환경
- URL: `http://localhost:3000/smart-analysis` (dev 서버 실행 상태)
- 브라우저: Chrome/Firefox/Safari 최신 버전
- 테스트 데이터: CSV 파일 준비

---

## ✅ 1단계: 데이터 업로드 - 에러 처리

### 1-1. 정상 파일 업로드
**목표**: 데이터가 있는 파일 업로드 성공

**절차**:
```
1. "1단계: 데이터 업로드" 카드 확인
2. CSV 파일 드래그 또는 "클릭해서 업로드" 클릭
3. 유효한 데이터 파일 선택
```

**기대 결과**:
- ✅ Alert 에러 메시지 없음
- ✅ 자동으로 "2단계: 기초 통계 분석"으로 전환
- ✅ 데이터셋 정보(행 수, 변수 수) 표시
- ✅ 변수 목록 표시 (숫자형 파란색, 범주형 초록색)

**증거 캡처**: 스크린샷 1-1

---

### 1-2. 빈 파일 업로드 (Optional)
**목표**: 데이터가 없는 파일 업로드 시 에러 메시지 표시

**절차**:
```
1. 빈 CSV 파일 또는 헤더만 있는 파일 준비
2. "1단계: 데이터 업로드"에서 업로드
```

**기대 결과**:
- ✅ 빨간색 Alert: "업로드된 데이터가 비어있습니다. 데이터가 있는 파일을 다시 업로드해주세요."
- ✅ 단계 전환 안 됨 (업로드 단계에 머물러 있음)
- ✅ FileUpload 컴포넌트 다시 표시

**증거 캡처**: 스크린샷 1-2

---

## ✅ 3단계: 가정 검정 - Tooltip 및 설명 텍스트

### 3-1. 정규성 검정 Tooltip
**목표**: "정규성 검정" 옆 Info 아이콘에 마우스 올려 설명 표시

**절차**:
```
1. "2단계: 기초 통계 분석"에서 "다음 단계: 가정 검정" 버튼 클릭
2. 약 3초 후 "3단계: 통계적 가정 검정" 표시
3. "정규성 검정 (Shapiro-Wilk Test)" 제목의 Info 아이콘 호버
```

**기대 결과**:
- ✅ 제목 옆 Info 아이콘 (?) 표시
- ✅ Info 아이콘에 마우스 올리면 검은색 Tooltip 팝업
- ✅ 텍스트: "데이터가 정규분포(종 모양)를 따르는지 검사합니다. p-값이 0.05보다 작으면 정규분포가 아닙니다."
- ✅ Tooltip이 제목 위에 중앙 정렬로 나타남

**증거 캡처**: 스크린샷 3-1a (마우스 전), 3-1b (마우스 호버)

---

### 3-2. p-값 해석 텍스트 - 정규분포
**목표**: p-값 옆에 "(정규분포)" 또는 "(정규분포 아님)" 표시 확인

**절차**:
```
1. "3단계: 통계적 가정 검정" 화면 보기
2. "정규성 검정" 섹션에서 각 변수의 p-값 확인
```

**기대 결과**:
- ✅ 각 변수마다 행: `[변수명] | p = 0.XXX (정규분포)` 또는 `p = 0.XXX (정규분포 아님)`
- ✅ p < 0.05면 "(정규분포 아님)" 표시
- ✅ p ≥ 0.05면 "(정규분포)" 표시
- ✅ 오른쪽 배지: 초록색 "정규분포 ✓" 또는 빨간색 "비정규분포 ✗"

**샘플 데이터**:
```
value1 | p = 0.234 (정규분포)  [정규분포 ✓]
value2 | p = 0.032 (정규분포 아님) [비정규분포 ✗]
```

**증거 캡처**: 스크린샷 3-2

---

### 3-3. 등분산성 검정 Tooltip
**목표**: "등분산성 검정" 옆 Info 아이콘에 마우스 올려 설명 표시

**절차**:
```
1. "3단계: 통계적 가정 검정" 화면의 아래쪽으로 스크롤
2. "등분산성 검정 (Levene's Test)" 제목의 Info 아이콘 호버
```

**기대 결과**:
- ✅ Info 아이콘 호버 시 검은색 Tooltip 팝업
- ✅ 텍스트: "각 그룹의 데이터 퍼짐정도가 같은지 검사합니다. p-값이 0.05보다 작으면 그룹 간 분산이 다릅니다."
- ✅ Tooltip이 제목 위에 중앙 정렬로 나타남

**증거 캡처**: 스크린샷 3-3a (마우스 전), 3-3b (마우스 호버)

---

### 3-4. p-값 해석 텍스트 - 등분산성
**목표**: 등분산성 p-값 옆에 "(분산 같음)" 또는 "(분산 다름)" 표시 확인

**절차**:
```
1. "3단계" 화면의 "등분산성 검정" 섹션 확인
2. 각 테스트의 p-값 해석 텍스트 확인
```

**기대 결과**:
- ✅ 각 테스트마다: `p = 0.XXX (분산 같음)` 또는 `p = 0.XXX (분산 다름)`
- ✅ p < 0.05면 "(분산 다름)" 표시
- ✅ p ≥ 0.05면 "(분산 같음)" 표시
- ✅ 오른쪽 배지: 초록색 "등분산 ✓" 또는 빨간색 "이분산 ✗"

**증거 캡처**: 스크린샷 3-4
