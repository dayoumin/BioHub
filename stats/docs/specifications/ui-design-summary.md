# 통계 분석 UI 설계 문서 (v2.0)

**작성일**: 2025-11-02
**버전**: 2.0 (사용자 맞춤형 대시보드)
**페이지**: `/statistics`

---

## 📊 개요

**사용자 맞춤형 대시보드 방식**의 통계 분석 인터페이스를 구현했습니다.

### 핵심 특징
- ✅ 4개 메인 탭 구조 (홈, 통계분석, 챗봇, 설정)
- ✅ 즐겨찾기 시스템 (localStorage 기반)
- ✅ 스마트 분석 기능 (AI 추천, 준비 중)
- ✅ 카드 기반 UI (버튼식 카테고리)
- ✅ 반응형 디자인 (모바일/태블릿/데스크톱)

---

## 🎯 핵심 설계 원칙

### 1. **사용자 맞춤형 대시보드**
- 홈 화면: 스마트 분석 + 즐겨찾기 통계 도구
- 개인화된 빠른 접근
- 최소 클릭으로 분석 시작

### 2. **직관적 네비게이션**
- 4개 탭으로 명확한 기능 분리
- 별표(⭐) 시스템으로 즐겨찾기 관리
- localStorage 기반 사용자 설정 유지

### 3. **점진적 기능 공개**
- 홈: 자주 쓰는 도구만 표시
- 통계분석: 모든 기능 카탈로그
- 챗봇/설정: 향후 확장 준비

---

## 🏗️ UI 구조

### **메인 탭 (4개)**

```
┌────────────────────────────────────────────────┐
│ [🏠 홈] [📊 통계분석] [🤖 챗봇] [⚙️ 설정]     │
└────────────────────────────────────────────────┘
```

#### 1. **홈 탭** (대시보드)
```
┌─────────────────────────────────┐
│      🎯 스마트 분석 (큰 버튼)      │ ← AI 추천 분석 (준비 중)
└─────────────────────────────────┘

┌─────────────────────────────────┐
│     ⭐ 내 통계 도구 (4열 그리드)    │ ← 즐겨찾기한 통계
│  [t-검정] [ANOVA] [회귀] [상관]  │
└─────────────────────────────────┘
```

**기능**:
- 스마트 분석: AI가 데이터 분석 후 최적의 통계 방법 추천
- 내 통계 도구: 별표로 즐겨찾기한 항목만 표시
- 빠른 시작: 2클릭으로 분석 시작

#### 2. **통계분석 탭** (전체 카탈로그)
```
┌─────────────────────────────────┐
│ 💡 별표를 클릭하면 홈에 추가됩니다  │
└─────────────────────────────────┘

[기술통계 카테고리 ──────────────]
  [평균/표준편차 ⭐] [빈도분석 ☆] ...

[가설검정 카테고리 ──────────────]
  [t-검정 ⭐] [ANOVA ⭐] ...

... (6개 카테고리)
```

**기능**:
- 6개 카테고리별 섹션
- 각 통계 카드에 별표(⭐) 버튼
- 별표 클릭 → localStorage 저장 → 홈 화면에 추가

#### 3. **챗봇 탭** (준비 중)
```
┌─────────────────────────────────┐
│      🤖 AI 통계 상담 (준비 중)     │
│  - 통계 방법 추천                │
│  - 데이터 분석 상담              │
│  - 실험 설계 조언                │
│  - 결과 해석 도움                │
└─────────────────────────────────┘
```

#### 4. **설정 탭** (기본 기능)
```
┌─────────────────────────────────┐
│      ⚙️ 설정 (준비 중)            │
│                                 │
│  빠른 작업:                      │
│  [모든 통계를 즐겨찾기에 추가]     │
│  [모든 즐겨찾기 해제]             │
└─────────────────────────────────┘
```

## 🔗 즐겨찾기 시스템

### 작동 원리
```typescript
// 1. 사용자가 별표 클릭
toggleFavorite('independent-t-test')

// 2. localStorage에 저장
localStorage.setItem('statisticsFavorites', JSON.stringify([
  'independent-t-test',
  'one-way-anova',
  'pearson-correlation'
]))

// 3. 홈 탭에 자동 표시
favoriteItems = allItems.filter(item =>
  favorites.includes(item.id)
)
```

### 데이터 구조
```json
{
  "statisticsFavorites": [
    "independent-t-test",
    "one-way-anova",
    "pearson-correlation",
    "linear-regression"
  ]
}
```

## 📊 사용자 경험 플로우

### 신규 사용자
```
1. 페이지 접속 → 홈 탭
2. "즐겨찾기한 통계가 없습니다" 메시지
3. [통계분석 보기] 버튼 클릭
4. 원하는 통계에 별표(⭐) 클릭
5. 홈 탭으로 돌아가면 즐겨찾기 표시
```

### 재방문 사용자
```
1. 페이지 접속 → 홈 탭
2. localStorage에서 즐겨찾기 로드
3. 내 통계 도구에서 [시작] 클릭
4. 즉시 분석 시작
```

### 통계 검색 사용자
```
1. 통계분석 탭 선택
2. 6개 카테고리 중 원하는 섹션으로 스크롤
3. 통계 카드 찾기
4. [분석 시작] 클릭
```
