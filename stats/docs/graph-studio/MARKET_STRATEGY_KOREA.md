## 2. 기존 문서에서 빠진 핵심 경쟁자 상세 분석

### 2-A. jamovi — 가장 빠르게 성장하는 직접 경쟁자 ★★★

> **핵심**: "SPSS를 무료로 쓰고 싶다"는 수요를 흡수. 그래프가 약해서 결국 Illustrator로 마무리.
> **BioHub 기회**: jamovi에서 분석 → BioHub에서 출판용 그림 만들기 워크플로 공식 지원.

| 항목 | jamovi | BioHub Graph Studio |
|------|--------|---------------------|
| 가격 | 무료/오픈소스 | 무료 |
| 플랫폼 | 앱 설치 필요 | 웹 브라우저 |
| 한국어 | ❌ 영어 전용 | ✅ |
| 그래프 품질 | 기본 수준, 커스터마이징 제한 | 출판용 목표 |
| 통계 범위 | 50+ 분석 | 43+ 메서드 |
| 저널 팔레트 | ❌ | ✅ NPG/AAAS/NEJM/Lancet/JAMA |
| 유의성 브래킷 | ❌ | ✅ 자동 스태킹 |
| CSV 내보내기 | ✅ | ← import 지원으로 연결 가능 |

**권장 전략**: jamovi의 CSV 내보내기 → BioHub 직접 import를 "공식 워크플로"로 문서화.
"jamovi로 분석하고 BioHub로 그림 만든다"는 투-툴 스택 포지셔닝.

---

### 2-B. Excel — 과소평가된 현실 최대 경쟁자 ★★★

한국 의학논문의 약 20-30%는 여전히 Excel 차트로 제출됩니다 (저널 심사 피드백 기준).

**Excel 차트의 치명적 문제점**:
- 에러바 정의 혼동 (SEM vs SD를 구분하지 않음)
- 저널 규격(mm) 미지원 — 픽셀 기반
- publication-ready 품질 불가 — 선 굵기, 폰트, 해상도
- 그래프 타입 제한 — violin, error-bar 없음
- 통계 자동화 없음

**전략**: Excel에서 이탈하는 사용자가 가장 큰 시장. 진입 장벽이 낮아야 함.
Graph Studio의 "Excel CSV 업로드 → 즉시 논문용 그래프" 플로는 이미 이 포지션.

### 2-C. MedCalc — 임상/의학 특화 유료 도구 ★★

| 항목 | MedCalc | BioHub |
|------|---------|--------|
| 가격 | €100+/년 | 무료 |
| ROC 분석 | ✅ 특화 (AUC + CI + Youden) | ❌ 미구현 |
| Survival (KM) | ✅ | ❌ 미구현 |
| Bland-Altman | ✅ | ❌ 미구현 |
| 한국어 | ❌ | ✅ |
| 웹 접근 | ❌ 설치 필요 | ✅ |
| 그래프 품질 | 기능적, 논문용은 미흡 | 출판용 목표 |
| 업데이트 빈도 | 낮음 | 높음 |

> **핵심 인사이트**: ROC + Kaplan-Meier 두 가지만 추가해도 한국 의학 연구자에게
> "무료 + 한국어 + 출판용 품질"로 MedCalc를 완전 대체 가능.

### 2-D. IBM SPSS — 지배적이지만 퇴조 중 ★

| 항목 | 내용 |
|------|------|
| 가격 | 학생용 ₩280,000+/년 |
| 한국 의학계 현황 | 여전히 표준 (순천향, 연세, 서울대 의대 커리큘럼) |
| 그래프 품질 | 매우 낮음 — 별도 도구 필수 |
| 쇠퇴 원인 | 비용 + jamovi/R 전환 증가 |
| BioHub 기회 | "SPSS 통계 + BioHub 그래프" 이분 워크플로 |

**현재 한국 의학 연구 표준 워크플로**:
```
데이터 분석 (SPSS) → 그래프 (Prism/Excel) → 최종 편집 (Illustrator/PPT) → 저널 제출
         ↓               ↓                        ↓
      ₩280k/년         $142/년                 $55k/년
```

**BioHub 목표 워크플로**:
```
데이터 분석 (SPSS or jamovi) → 그래프+내보내기 (BioHub) → 저널 제출
              ↓                        ↓
         ₩280k or 무료                 무료
```

### 3-B. 한국 의학/임상 연구 — 가장 큰 미개척 시장

한국은 세계적 수준의 임상 연구 인프라를 보유:
- 삼성서울병원, 서울아산병원, 세브란스, 서울성모병원 (빅4 연구중심병원)
- 연간 임상 SCI 논문 약 20,000편 이상 (2024년 기준)

**임상 연구자가 가장 많이 쓰는 그래프 TOP 5 (현재 BioHub 지원 여부)**:

| 순위 | 그래프 | 현재 | 난이도 | 전략 우선순위 |
|------|--------|------|--------|------------|
| 1 | ROC Curve (AUC + CI + Cut-off) | ❌ | 중간 | **P0** |
| 2 | Kaplan-Meier 생존곡선 (log-rank p값) | ❌ | 높음 | **P0** |
| 3 | Forest Plot (OR/HR + 95% CI) | ❌ | 중간 | **P1** |
| 4 | 막대 + 에러바 + 유의성 | ✅ G2 완료 | — | 완료 |
| 5 | Bland-Altman (측정 일치도) | ❌ | 낮음 | **P2** |

> **임팩트**: ROC + KM 두 개만 구현해도 한국 임상 의학 논문 작성자의 60% 이상에게
> 직접적 가치 제공. MedCalc(€100/년) 완전 대체 가능.

### 3-C. 한국 연구 지원 체계 특화

**NRF(한국연구재단) 보고서 표준**

정부 연구비(NRF, KEIT, KHIDI) 과제 결과 보고서의 그림은 특정 양식이 있으나
현재 어떤 도구도 이를 지원하지 않음.

```
NRF 성과보고서 그림 요건 (예시):
- 해상도: 300 DPI 이상
- 파일형식: PNG 또는 TIFF
- 제목 포함 시 폰트: 맑은 고딕 또는 Arial
- 색상: CMYK 또는 RGB (인쇄용은 CMYK)
```

**제안**: "NRF 보고서용" 내보내기 프리셋 (낮은 구현 난이도, 높은 마케팅 가치).

---

### 3-D. 한국어 통계 해석 텍스트 자동 생성 — 독보적 차별화

현재 경쟁 도구 어디서도 제공하지 않는 기능.
BioHub Smart Flow에 이미 AI 해석이 있으므로 Graph Studio와 연결 가능.

**예시 자동 생성 텍스트**:
```
"두 군 간 체중의 차이는 통계적으로 유의하였다
 (독립 t-검정: t = 3.42, df = 48, p = 0.001, Cohen's d = 0.96)."

"수술 후 생존율은 A군에서 유의하게 높았다
 (Kaplan-Meier log-rank: χ² = 8.34, p = 0.004)."
```

이는 "그림 + 방법 서술"을 동시에 생성하는 것으로, 논문 작성 시간을 크게 단축.

### 3-F. 수산·농업 데이터 — BioHub 원래 강점 영역 (니치)

BioHub의 기원이 수산 데이터 분석인 만큼, 이 분야에서의 특화는 자연스러운 차별화.

| 특화 기능 | 대상 | 현재 경쟁 |
|---------|------|---------|
| 어류 성장 곡선 (von Bertalanffy) | 수산과학원(NIFS), 수산 연구자 | R 코드만 가능 |
| 수온-성장 산점도 프리셋 | 양식 연구 | 없음 |
| NIFS 데이터 포맷 자동 인식 | 수산 연구 기관 | 없음 |
| 농진청(RDA) 데이터 형식 지원 | 농업 연구 | 없음 |

## 5. 포지셔닝 결론 — "시장 공백의 유일한 점유자"

```
현재 시장 공백:
┌─────────────────────────────────────────────────────────┐
│  "통계 분석 + 출판용 그래프 + 한국어 + 무료"             │
│   → 이 조합을 제공하는 도구가 현재 존재하지 않음          │
│                                                         │
│  jamovi   : 통계 ✅ / 그래프 ❌ / 한국어 ❌               │
│  Prism    : 통계 ✅ / 그래프 ✅ / 한국어 ❌ / $142       │
│  SPSS     : 통계 ✅ / 그래프 ❌(논문용) / 한국어 부분     │
│  MedCalc  : 통계 ✅(임상) / 그래프 △ / 한국어 ❌ / €100 │
│  Excel    : 통계 △ / 그래프 ❌(논문용) / 한국어 ✅       │
│                                                         │
│  ★ BioHub : 통계 ✅ / 그래프 ✅(성장중) / 한국어 ✅ / 무료│
└─────────────────────────────────────────────────────────┘
```

**단기 목표**: ROC + KM 추가 → MedCalc 완전 대체
**중기 목표**: jamovi + BioHub 공식 워크플로 → SPSS 이탈자 흡수
**장기 목표**: 한국어 AI 해석 + 저널 자동 포맷 → 한국 연구자 표준 도구
