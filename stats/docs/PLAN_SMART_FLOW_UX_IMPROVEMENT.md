# Smart Flow UX 개선 계획

## 현재 문제점

### 1. 스크롤 문제
- Step 1에서 하단으로 스크롤이 안 됨 (확인 필요)

### 2. 중복된 데이터 요약 표시
- **Step 1 (DataValidationStep)**: 데이터 준비 완료 카드 (표본 크기, 변수, 품질, 파일명)
- **Step 3 (PurposeInputStep)**: DataProfileSummary 컴포넌트 (표본 크기, 변수, 품질, 권장분석)
- → 동일한 정보가 2번 표시됨

### 3. Step 2 전체 데이터 테이블
- 현재: 항상 하단에 전체 데이터 테이블 표시 (스크롤 필요)
- 문제: 페이지가 너무 길어지고, 항상 필요한 정보가 아님

---

## 개선 계획

### Step 1: 업로드 (간소화)
**목표**: 파일 업로드 + 기본 검증만

**표시 내용**:
- 드래그앤드롭 업로드 영역
- 업로드 완료 시:
  - ✓ 업로드 완료: 파일명
  - 기본 정보만: `30행 × 6열` (한 줄)
  - "다음 단계" 버튼

**제거**:
- ❌ DataValidationStep의 상세 카드 (표본 크기, 변수, 품질, 파일명)
- ❌ 분석 추천 카드 (Step 3으로 이동)
- ❌ 변수 요약 테이블 (Step 2로 이동)

---

### Step 2: 탐색 (데이터 품질 + 탐색)
**목표**: 데이터 품질 확인 + 변수 탐색

**표시 내용**:
1. **데이터 요약 카드** (Step 1에서 이동)
   - 표본 크기, 변수 타입, 결측값, 품질
   - 분석 추천 목록

2. **기초 통계량** (현재 유지)

3. **통계적 가정 검증** (현재 유지)

4. **데이터 분포 시각화** (현재 유지)

5. **산점도 / 상관계수 행렬** (현재 유지)

6. **전체 데이터 보기** (변경)
   - 기존: 항상 표시되는 큰 테이블
   - 변경: "전체 데이터 보기" 링크/버튼 → 새 창 또는 모달로 표시

---

### Step 3: 방법 (AI 추천)
**목표**: 분석 목적 선택 + AI 추천

**표시 내용**:
- 분석 목적 선택 카드 (5개)
- AI 추천 결과

**제거**:
- ❌ DataProfileSummary (Step 2에서 이미 확인함, 중복 제거)
