## 1. 변수 역할 이해하기

### 🎯 **핵심 개념: "무엇을 설명하고, 무엇으로 설명하는가?"**

통계 분석에서 변수는 **역할(Role)**에 따라 구분됩니다. 가장 중요한 두 가지는:

| 역할 | 설명 | 예시 |
|-----|------|------|
| **종속변수 (Dependent Variable)** | **설명하려는 대상** (결과, Y) | 넙치 체중, 생산량, 생존율 |
| **독립변수 (Independent Variable)** | **설명에 사용하는 변수** (원인, X) | 사료 종류, 수온, 염분도 |

**🔍 예시 1: 사료 종류가 넙치 체중에 미치는 영향**
- **종속변수**: 넙치 체중 (우리가 알고 싶은 것)
- **독립변수**: 사료 종류 (A, B, C)

**🔍 예시 2: 수온이 넙치 생장률에 미치는 영향**
- **종속변수**: 생장률 (%/day)
- **독립변수**: 수온 (°C)

---

### 📋 **전체 변수 역할 목록**

| 역할 | 한글 이름 | 사용 예시 | 주요 통계 방법 |
|-----|----------|----------|--------------|
| **dependent** | 종속변수 | 체중, 생산량, 생존율 | 거의 모든 분석 |
| **independent** | 독립변수 | 사료섭취량, 수온, 염분도 | 회귀분석, 상관분석 |
| **factor** | 요인 (그룹 변수) | 사료 종류, 성별, 지역 | ANOVA, t-검정 |
| **covariate** | 공변량 (통제변수) | 초기체중, 초기체장 | ANCOVA |
| **within** | 개체내 요인 | 측정 시점 (1일, 7일, 14일) | 반복측정 ANOVA |
| **blocking** | 블록 변수 | 수조번호, 양식장 | 블록 설계 |

## 2. 변수 타입 이해하기

### 🔢 **변수 타입 구분 (6가지)**

| 타입 | 설명 | 예시 | 허용 통계 |
|-----|------|------|----------|
| **continuous** | 연속형 (실수) | 체중(150.5g), 수온(25.3°C) | t-검정, ANOVA, 회귀분석 |
| **categorical** | 범주형 (문자열) | 사료 종류 (A, B, C), 양식장 (1, 2, 3) | ANOVA, 카이제곱 검정 |
| **binary** | 이진형 (2개 값) | 성별 (남/여), 생존 (Yes/No) | t-검정, 로지스틱 회귀 |
| **ordinal** | 서열형 (순서 있음) | 만족도 (1~5점), 학력 (초/중/고/대) | 비모수 검정 (Kruskal-Wallis) |
| **count** | 카운트 (양의 정수) | 치어 수 (10마리), 폐사 수 | Poisson 회귀 |
| **date** | 날짜/시간 | 2024-01-01, 14:30:00 | 시계열 분석 |

---

### ⚙️ **변수 타입 자동 감지**

플랫폼은 데이터를 업로드하면 **자동으로 변수 타입을 감지**합니다:

| 데이터 예시 | 자동 감지 타입 |
|------------|---------------|
| 150.5, 200.3, 175.8 | **continuous** |
| "A", "B", "C" | **categorical** |
| "남", "여" | **binary** |
| 1, 2, 3, 4, 5 (정수) | **count** 또는 **ordinal** |

**⚠️ 주의**: 정수는 **count**로 감지되지만, 순서형(1~5점)이면 수동으로 **ordinal**로 변경하세요!

## 3. 통계 방법별 변수 선택 가이드

### 📊 **1) 기술통계 (Descriptive Statistics)**

#### **기술통계량 (Descriptive Stats)**
- **목적**: 평균, 표준편차, 최솟값/최댓값 계산
- **변수 요구사항**:
  - 종속변수 (dependent): 1개 이상, 연속형
  - 그룹 변수 (factor): 선택적, 범주형
- **예시**:
  ```
  종속변수: 넙치 체중 (continuous)
  그룹 변수: 사료 종류 (categorical) - 선택
  ```

#### **빈도분석 (Frequency Table)**
- **목적**: 범주별 빈도와 비율 계산
- **변수 요구사항**:
  - 분석 변수 (dependent): 1개 이상, 범주형/이진형
- **예시**:
  ```
  분석 변수: 성별 (binary), 학력 (categorical)
  ```

---

### 🔬 **2) 평균 비교 (Compare Means)**

#### **독립표본 t-검정 (Two-Sample t-test)**
- **목적**: 두 그룹 간 평균 비교
- **변수 요구사항**:
  - 종속변수 (dependent): 1개, 연속형
  - 그룹 변수 (factor): 1개, **정확히 2개 수준** (예: 남/여)
- **예시**:
  ```
  종속변수: 넙치 체중 (continuous)
  그룹 변수: 성별 (binary) - "암", "수"
  ```
- **⚠️ 주의**: 그룹 변수가 3개 이상이면 **일원분산분석(One-Way ANOVA)** 사용!

#### **대응표본 t-검정 (Paired t-test)**
- **목적**: 동일 대상의 전후 비교
- **변수 요구사항**:
  - 대응 변수 (dependent): **정확히 2개**, 연속형
  - 순서 중요: 첫 번째 - 두 번째 = 차이값
- **예시**:
  ```
  변수 1: 사전체중_g (continuous)
  변수 2: 사후체중_g (continuous)
  ```

#### **일원분산분석 (One-Way ANOVA)**
- **목적**: 3개 이상 그룹 간 평균 비교
- **변수 요구사항**:
  - 종속변수 (dependent): 1개, 연속형
  - 요인 (factor): 1개, **3개 이상 수준** (예: A, B, C)
- **예시**:
  ```
  종속변수: 넙치 체중 (continuous)
  요인: 사료 종류 (categorical) - "A", "B", "C"
  ```

---

### 📈 **3) 회귀분석 (Regression Analysis)**

#### **단순회귀분석 (Simple Regression)**
- **목적**: 1개 독립변수로 종속변수 예측
- **변수 요구사항**:
  - 종속변수 (dependent): 1개, 연속형
  - 독립변수 (independent): 1개, 연속형
- **예시**:
  ```
  종속변수: 체중 (continuous)
  독립변수: 사료량 (continuous)
  ```

#### **다중회귀분석 (Multiple Regression)**
- **목적**: 2개 이상 독립변수로 종속변수 예측
- **변수 요구사항**:
  - 종속변수 (dependent): 1개, 연속형
  - 독립변수 (independent): 2개 이상, 연속형
- **예시**:
  ```
  종속변수: 생산량_kg (continuous)
  독립변수: 사료섭취량_g (continuous), 수온_C (continuous), pH (continuous)
  ```

#### **로지스틱 회귀분석 (Logistic Regression)**
- **목적**: 이진형 결과 예측 (Yes/No, 생존/사망)
- **변수 요구사항**:
  - 종속변수 (dependent): 1개, **이진형** (0/1, Yes/No)
  - 독립변수 (independent): 1개 이상, 연속형 또는 범주형
- **예시**:
  ```
  종속변수: 생존 여부 (binary) - "생존", "폐사"
  독립변수: 수온_C (continuous), 염분도_ppt (continuous)
  ```

---

### 🔗 **4) 상관분석 (Correlation Analysis)**

#### **Pearson 상관분석**
- **목적**: 두 연속형 변수 간 선형 관계 측정
- **변수 요구사항**:
  - 변수 (dependent): 2개 이상, **연속형**
- **예시**:
  ```
  변수: 체중_g (continuous), 체장_cm (continuous), 비만도 (continuous)
  ```

#### **Spearman 상관분석**
- **목적**: 서열형 변수 간 관계 측정
- **변수 요구사항**:
  - 변수 (dependent): 2개 이상, **서열형** 또는 연속형
- **예시**:
  ```
  변수: 만족도 (ordinal, 1~5), 선호도 (ordinal, 1~5)
  ```

---

### 🎲 **5) 비모수 검정 (Nonparametric Tests)**

#### **Mann-Whitney U 검정**
- **목적**: 정규성 가정하지 않는 두 그룹 비교 (t-검정 대체)
- **변수 요구사항**:
  - 종속변수 (dependent): 1개, 연속형 또는 서열형
  - 그룹 변수 (factor): 1개, **정확히 2개 수준**
- **예시**:
  ```
  종속변수: 만족도 (ordinal, 1~5)
  그룹 변수: 성별 (binary)
  ```

#### **Kruskal-Wallis 검정**
- **목적**: 정규성 가정하지 않는 3개 이상 그룹 비교 (ANOVA 대체)
- **변수 요구사항**:
  - 종속변수 (dependent): 1개, 연속형 또는 서열형
  - 요인 (factor): 1개, **3개 이상 수준**
- **예시**:
  ```
  종속변수: 만족도 (ordinal)
  요인: 제품 종류 (categorical) - "A", "B", "C", "D"
  ```

---

### ✅ **6) 카이제곱 검정 (Chi-Square Tests)**

#### **독립성 검정 (Chi-Square Independence)**
- **목적**: 두 범주형 변수 간 독립성 검정
- **변수 요구사항**:
  - 행 변수 (independent): 1개, 범주형
  - 열 변수 (dependent): 1개, 범주형
- **예시**:
  ```
  행 변수: 성별 (binary)
  열 변수: 흡연 여부 (binary)
  ```

## 4. 자주 묻는 질문 (FAQ)

### ❓ **Q1: 변수 타입을 잘못 선택하면 어떻게 되나요?**
**A**: 분석 시작 시 **자동으로 경고 메시지**가 표시됩니다.
- 예: "종속변수는 연속형이어야 하지만 범주형이 선택되었습니다."
- **해결 방법**: 변수 선택 화면에서 다시 선택하거나, 데이터 변환 후 재업로드

---

### ❓ **Q2: "종속변수"와 "독립변수"를 헷갈려요!**
**A**: 다음 질문으로 구분하세요:
- **종속변수**: "무엇을 알고 싶은가?" → 결과 (Y)
- **독립변수**: "무엇이 영향을 주는가?" → 원인 (X)

**예시**: "사료 종류가 넙치 체중에 미치는 영향"
- 알고 싶은 것: 넙치 체중_g → **종속변수**
- 영향을 주는 것: 사료 종류 (A, B, C) → **독립변수**

---

### ❓ **Q3: t-검정과 ANOVA를 언제 사용하나요?**
**A**: **그룹 수**로 구분하세요:
- **2개 그룹**: t-검정 (예: 암 vs 수)
- **3개 이상 그룹**: ANOVA (예: 사료 A, B, C)

---

### ❓ **Q4: 변수가 자동 감지되지 않으면?**
**A**: 데이터 형식을 확인하세요:
1. CSV 파일이 UTF-8 인코딩인지 확인
2. 첫 번째 행이 변수명인지 확인
3. 결측치가 빈칸 또는 "NA"로 표시되었는지 확인

---

### ❓ **Q5: 연속형 변수를 범주형으로 변환하고 싶어요!**
**A**: 데이터 전처리 도구 사용:
1. 대시보드에서 **"데이터 변환"** 메뉴 선택
2. "변수 변환" → "연속형 → 범주형"
3. 구간 설정 (예: 0-50 → "낮음", 51-100 → "높음")

---

### ❓ **Q6: "요인(factor)"과 "독립변수(independent)"의 차이는?**
**A**: 변수 타입으로 구분:
- **요인(factor)**: 범주형 변수 (예: 사료 종류 A, B, C) → ANOVA
- **독립변수(independent)**: 연속형 변수 (예: 사료량 100g, 200g) → 회귀분석

---

### ❓ **Q7: 한 번에 여러 종속변수를 선택할 수 있나요?**
**A**: 통계 방법에 따라 다릅니다:
- **가능**: 기술통계, 상관분석, MANOVA
- **불가능**: t-검정, ANOVA, 회귀분석 (1개만)
