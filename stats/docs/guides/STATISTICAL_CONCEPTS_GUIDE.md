## 1. 가설 검정 기초

### 1.1 귀무가설 vs 대립가설

**귀무가설 (H₀, Null Hypothesis)**:
- "차이가 없다" 또는 "관계가 없다"는 가정
- 통계 검정으로 **기각하려는** 가설
- 예시: "신약 효과 = 기존약 효과"

**대립가설 (H₁, Alternative Hypothesis)**:
- 연구자가 **증명하려는** 가설
- 귀무가설의 반대
- 예시: "신약 효과 ≠ 기존약 효과"

---

### 1.2 p-value의 의미

**정의**:
> p-value = 귀무가설이 참일 때, 현재 관측된 결과보다 극단적인 결과가 나올 확률

**해석**:
- **p < 0.05**: 귀무가설 기각 → "통계적으로 유의한 차이/관계 있음"
- **p ≥ 0.05**: 귀무가설 채택 → "통계적으로 유의한 차이/관계 없음"

**⚠️ 주의사항**:
- p-value는 "효과의 크기"가 아닙니다
- p-value는 "결과의 중요성"이 아닙니다
- p < 0.05라고 항상 "실질적 의미"가 있는 건 아닙니다

**예시**:
```
수온 변화가 어류 성장에 미치는 영향 연구
- p = 0.03 → "수온 변화와 성장률 간 통계적으로 유의한 관계 있음"
- p = 0.15 → "통계적으로 유의한 관계 없음" (하지만 실제로는 있을 수도!)
```

---

### 1.3 유의수준(α)이란?

**정의**:
> α = 귀무가설이 참인데 기각하는 오류(제1종 오류)를 허용하는 최대 확률

**판정 기준**:
```
p-value < α → 귀무가설 기각 (유의함)
p-value ≥ α → 귀무가설 채택 (유의하지 않음)
```

**표준 기준**:
- **α = 0.05** (5%): 대부분의 연구 (표준)
- **α = 0.01** (1%): 의학/생명과학 (엄격)
- **α = 0.10** (10%): 탐색적 연구 (관대)

## 2. 양측 검정 vs 단측 검정

### 2.1 양측 검정 (Two-tailed Test)

**연구 질문**: "A와 B가 **다른가**?"

**가설**:
```
H₀: μ_A = μ_B
H₁: μ_A ≠ μ_B  (A > B 또는 A < B 둘 다 포함)
```

**특징**:
- ✅ **방향 무관**: 증가/감소 모두 탐지
- ✅ **안전성**: 나쁜 쪽 변화도 감지 가능
- ✅ **과학적 엄격성**: 학계 표준 (95%+ 사용)

**p-value 계산**:
```
p_양측 = P(|T| ≥ |t_관측값|)
      = 2 × P(T ≥ |t_관측값|)  (대칭 분포 가정)
```

**실무 예시**:
```
수산과학 연구: "수온 변화가 어류 성장에 영향을 주는가?"
→ 증가할지, 감소할지 모름 → 양측 검정

결과:
- 25°C: 평균 체중 100g
- 30°C: 평균 체중 90g
- p = 0.01 (양측)
→ "수온 변화가 성장에 유의한 영향을 줌" (감소!)
```

---

### 2.2 단측 검정 (One-tailed Test)

**연구 질문**: "A가 B보다 **큰가**?" 또는 "A가 B보다 **작은가**?"

**가설 (Greater)**:
```
H₀: μ_A ≤ μ_B
H₁: μ_A > μ_B  (A가 B보다 클 때만 관심)
```

**가설 (Less)**:
```
H₀: μ_A ≥ μ_B
H₁: μ_A < μ_B  (A가 B보다 작을 때만 관심)
```

**특징**:
- ✅ **검정력 증가**: 같은 α에서 더 민감 (양측의 2배)
- ⚠️ **방향 고정**: 한쪽 방향만 검정
- ⚠️ **위험성**: 반대 방향 변화 놓칠 수 있음
- ⚠️ **드물게 사용**: 전체의 5% 미만

**p-value 계산**:
```
p_단측(greater) = P(T ≥ t_관측값)
                = p_양측 / 2  (t_관측값 > 0일 때)

p_단측(less) = P(T ≤ t_관측값)
             = p_양측 / 2  (t_관측값 < 0일 때)
```

**실무 예시**:
```
품질 관리: "새 공정이 불량률을 낮추는가?"
→ 감소만 관심 (증가는 어차피 채택 안 함) → 단측(less)

결과:
- 기존: 불량률 5%
- 새로운: 불량률 3%
- p = 0.02 (단측, less)
→ "새 공정이 불량률을 유의하게 낮춤"

⚠️ 주의: 만약 불량률이 7%로 증가했다면?
→ 단측(less): p = 0.98 → "개선 아님" (당연)
→ 하지만 "악화"했다는 사실을 명시 못 함!
```

---

### 2.3 양측 vs 단측: 언제 어떤 검정?

#### **양측 검정을 쓰는 경우** (95%+)

| 상황 | 예시 |
|------|------|
| **탐색적 연구** | "수온과 성장률 간 관계가 있는가?" |
| **안전성 고려** | "신약이 기존약과 다른가?" (나쁠 수도!) |
| **방향 모름** | "성별에 따라 점수가 다른가?" |
| **학계 표준** | 대부분의 저널이 양측 요구 |
| **잘 모르겠을 때** | **양측 선택이 안전함!** |

#### **단측 검정을 쓰는 경우** (5% 미만)

| 상황 | 예시 | 주의사항 |
|------|------|----------|
| **방향 명확** | "품질 개선 입증" | 사전 정당화 필요 |
| **검정력 증가 필요** | "비열등성 검정" | 리뷰어 승인 필수 |
| **과거 연구 확립** | "이론적으로 한쪽만 가능" | 사전 등록 권장 |

---

### 2.4 p-value 비교 (동일 데이터)

**데이터**:
```
Group A: [10, 12, 14, 16, 18] (평균 14)
Group B: [8, 9, 10, 11, 12]   (평균 10)
t_관측값 = 3.2 (양수, A > B)
```

**결과 비교**:

| 검정 유형 | p-value | α=0.05 판정 | 해석 |
|-----------|---------|-------------|------|
| **양측** | 0.012 | 유의 ✅ | "A와 B의 평균이 다름" |
| **단측 (A > B)** | 0.006 | 유의 ✅ | "A가 B보다 큼" |
| **단측 (A < B)** | 0.994 | 비유의 ❌ | "A가 B보다 작지 않음" |

**핵심**:
- 같은 데이터라도 검정 유형에 따라 p-value가 **다름**
- 단측이 양측보다 p-value가 작음 (검정력 2배)
- 하지만 단측은 **반대 방향을 놓칠 위험** 있음!

---

### 2.5 학계/업계 관행

#### **양측 검정을 요구하는 분야**:

1. **의학/생명과학**
   - FDA, EMA (유럽의약품청)
   - 임상시험 가이드라인
   - 이유: 안전성 (악화 탐지 필수)

2. **심리학/사회과학**
   - APA (미국심리학회) 스타일
   - 대부분의 저널
   - 이유: 과학적 엄격성

3. **수산과학/생태학**
   - 대부분의 연구
   - 이유: 탐색적 (방향 모름)

#### **단측 검정을 허용하는 분야** (사전 정당화 필수):

1. **품질 관리/공학**
   - 6 Sigma, 품질 개선
   - 이유: 개선 입증 목적 명확

2. **임상시험 (특수 케이스)**
   - 비열등성 검정 (Non-inferiority)
   - 우월성 검정 (Superiority)
   - 이유: 사전에 방향 명시

## 3. 유의수준(α) 선택 가이드

### 3.1 α = 0.05 (표준, 95%+ 사용)

**특징**:
- ✅ 학계 표준 (가장 널리 사용)
- ✅ 대부분의 저널이 인정
- ✅ 제1종 오류 5% 허용

**사용 분야**:
- 사회과학 (심리학, 경제학)
- 수산과학, 생태학
- 일반 학술 연구

**해석**:
```
p = 0.04 → 유의함 ✅ (5% 기준 통과)
p = 0.06 → 유의하지 않음 ❌ (5% 기준 미달)
```

---

### 3.2 α = 0.01 (엄격, 의학/생명과학)

**특징**:
- ✅ 매우 엄격한 기준
- ✅ 안전성 중요 시
- ✅ 제1종 오류 1%만 허용

**사용 분야**:
- 의학 연구 (임상시험)
- 생명과학 (유전자 연구)
- 고위험 연구 (안전성 중요)

**해석**:
```
p = 0.008 → 유의함 ✅ (1% 기준 통과)
p = 0.023 → 유의하지 않음 ❌ (1% 기준 미달, 하지만 5%로는 유의!)
```

**장점**:
- 잘못된 양성 결과 감소 (신뢰성 증가)

**단점**:
- 검정력 감소 (실제 효과 놓칠 가능성 증가)

---

### 3.3 α = 0.10 (관대, 탐색적 연구)

**특징**:
- ✅ 탐색적 연구
- ✅ 파일럿 연구
- ✅ 제1종 오류 10% 허용

**사용 분야**:
- 예비 연구 (Pilot Study)
- 가설 생성 연구
- 경향성 탐색

**해석**:
```
p = 0.08 → 유의함 ✅ (10% 기준 통과)
p = 0.15 → 유의하지 않음 ❌ (10% 기준 미달)
```

**장점**:
- 검정력 증가 (실제 효과 더 잘 탐지)

**단점**:
- 잘못된 양성 결과 증가 (신뢰성 감소)
- 학계 논문 게재 어려움 (비표준)

---

### 3.4 유의수준 비교 (동일 데이터)

**데이터**: p-value = 0.023

| α | 판정 | 해석 |
|---|------|------|
| **0.10** | 유의 ✅ | "유의한 차이 있음 (p=0.023, α=0.10)" |
| **0.05** | 유의 ✅ | "유의한 차이 있음 (p=0.023, α=0.05)" |
| **0.01** | 비유의 ❌ | "유의한 차이 없음 (p=0.023, α=0.01)" |

**핵심**:
- 같은 p-value도 α에 따라 **결론이 달라짐**
- α 선택은 연구 분야/목적에 따라 결정
- **사전에 결정하고 사전 등록** 권장 (p-hacking 방지)

---

### 3.5 α 선택 의사결정 트리

```
Q1: 연구 분야가 의학/생명과학인가?
├─ Yes → α = 0.01 (엄격)
└─ No → Q2로

Q2: 탐색적 연구/파일럿 연구인가?
├─ Yes → α = 0.10 (관대)
└─ No → α = 0.05 (표준) ✅ 대부분 여기!

⚠️ 주의: 결과 보고 후 α 변경 금지 (p-hacking)
```

## 4. 효과 크기 해석

### 4.1 효과 크기란?

**정의**:
> 효과 크기 = 집단 간 차이 또는 관계의 **실질적 크기**를 나타내는 지표

**p-value와의 차이**:
- **p-value**: "차이가 우연인가?" (통계적 유의성)
- **효과 크기**: "차이가 얼마나 큰가?" (실질적 중요성)

**예시**:
```
연구 A: n=1000, 차이=1점, p=0.001 (유의!) → 효과 크기: 0.05 (매우 작음)
연구 B: n=30,   차이=10점, p=0.08 (비유의) → 효과 크기: 0.80 (큼!)

→ A는 통계적으로 유의하지만 실질적 의미 없음
→ B는 비유의하지만 실질적으로 의미 있을 수 있음 (검정력 부족)
```

---

### 4.2 Cohen's d (두 집단 평균 차이)

**계산**:
```
d = (평균₁ - 평균₂) / 통합 표준편차
```

**해석 기준** (Cohen, 1988):

| 범위 | 해석 | 실무 의미 |
|------|------|-----------|
| **d < 0.2** | 매우 작은 효과 | 거의 차이 없음 |
| **0.2 ≤ d < 0.5** | 작은 효과 | 작지만 의미 있을 수 있음 |
| **0.5 ≤ d < 0.8** | 중간 효과 | 명확한 차이 |
| **d ≥ 0.8** | 큰 효과 | 매우 큰 차이 |

**⚠️ 주의**: 분야별로 기준 다름!
- **의학**: d=0.3 → "큼" (생명 관련)
- **심리학**: d=0.5 → "중간"
- **물리학**: d=0.5 → "매우 작음" (정밀 측정)

**예시**:
```
수산과학 연구: 사료 종류에 따른 성장률 비교
- 사료 A: 평균 100g, SD=10
- 사료 B: 평균 115g, SD=12
- d = (115-100) / 11 = 1.36 (매우 큰 효과!)
→ "사료 B가 성장률을 크게 증가시킴"
```

---

### 4.3 Pearson's r (상관 효과 크기)

**범위**: -1 ≤ r ≤ 1

**해석 기준**:

| 범위 | 해석 | 의미 |
|------|------|------|
| **\|r\| < 0.1** | 거의 없음 | 무시 가능한 상관 |
| **0.1 ≤ \|r\| < 0.3** | 약한 상관 | 작지만 의미 있을 수 있음 |
| **0.3 ≤ \|r\| < 0.5** | 중간 상관 | 명확한 관계 |
| **\|r\| ≥ 0.5** | 강한 상관 | 매우 강한 관계 |

**예시**:
```
수온과 성장률 상관 연구
- r = 0.65 (강한 양의 상관)
- p = 0.002 (유의)
→ "수온이 높을수록 성장률이 크게 증가함"

수질과 질병 발생률
- r = -0.15 (약한 음의 상관)
- p = 0.03 (유의)
→ "통계적으로 유의하지만 실질적 관계는 약함"
```

---

### 4.4 η² (Eta-squared, ANOVA 효과 크기)

**정의**:
> η² = 집단 간 분산 / 총 분산
> (집단 차이가 전체 변동의 몇 %를 설명하는가?)

**해석 기준**:

| 범위 | 해석 | 설명력 |
|------|------|--------|
| **η² < 0.01** | 매우 작음 | 1% 미만 |
| **0.01 ≤ η² < 0.06** | 작음 | 1-6% |
| **0.06 ≤ η² < 0.14** | 중간 | 6-14% |
| **η² ≥ 0.14** | 큼 | 14% 이상 |

**예시**:
```
3개 사료 종류에 따른 성장률 ANOVA
- F(2, 87) = 8.45, p = 0.001
- η² = 0.16 (큰 효과)
→ "사료 종류가 성장률 변동의 16%를 설명함"
```

---

### 4.5 효과 크기 보고 가이드

**APA 스타일 권장**:
```
독립표본 t-검정:
"사료 A (M = 100, SD = 10)와 사료 B (M = 115, SD = 12) 간
유의한 차이가 있었다, t(58) = 5.21, p < .001, d = 1.36."

상관 분석:
"수온과 성장률 간 강한 양의 상관이 있었다,
r(48) = .65, p = .002."

ANOVA:
"사료 종류에 따른 성장률 차이가 유의했다,
F(2, 87) = 8.45, p = .001, η² = .16."
```

**핵심**:
- ✅ p-value + 효과 크기 **모두** 보고
- ✅ 통계적 유의성 ≠ 실질적 중요성
- ✅ 효과 크기로 실무 의미 판단

---

## 5. 실무 사례 연구

### 5.1 수산과학: 수온과 어류 성장

**연구 질문**:
> "수온 변화(25°C vs 30°C)가 어류 성장률에 영향을 주는가?"

**연구 설계**:
- 독립변수: 수온 (25°C, 30°C)
- 종속변수: 60일 후 체중 증가량 (g)
- 표본 크기: 각 30마리

**결과**:
```
25°C: 평균 100g, SD = 15g
30°C: 평균 90g,  SD = 18g
t(58) = 2.45, p = 0.018 (양측)
Cohen's d = 0.61 (중간 효과)
```

**해석**:
```
✅ 양측 검정 선택 이유:
   - 수온 증가가 성장을 촉진할지, 억제할지 모름
   - 탐색적 연구 (기존 연구 부족)

✅ α = 0.05 선택 이유:
   - 수산과학 표준
   - 학술 논문 게재 대상

✅ 결론:
   "수온 30°C가 25°C보다 성장률을 유의하게 감소시켰다
   (p = 0.018, d = 0.61). 중간 크기의 효과로,
   실무적으로 의미 있는 차이이다."

⚠️ 만약 단측(greater)으로 했다면?
   → p = 0.991 (비유의)
   → "수온 증가가 성장을 촉진하지 않음"
   → 하지만 실제로는 "억제"했는데 놓침!
```

---

### 5.2 의학 연구: 신약 효능 임상시험

**연구 질문**:
> "신약이 기존약보다 혈압 강하 효과가 다른가?"

**연구 설계**:
- 독립변수: 약물 종류 (신약, 기존약)
- 종속변수: 8주 후 수축기 혈압 감소량 (mmHg)
- 표본 크기: 각 150명

**결과**:
```
신약:   평균 -18 mmHg, SD = 8 mmHg
기존약: 평균 -15 mmHg, SD = 7 mmHg
t(298) = 3.21, p = 0.002 (양측)
Cohen's d = 0.39 (작은 효과)
```

**해석**:
```
✅ 양측 검정 선택 이유:
   - 신약이 더 좋을 수도, 나쁠 수도 있음 (안전성 고려)
   - FDA 가이드라인 요구

✅ α = 0.01 선택 이유:
   - 임상시험 (생명 관련, 엄격 기준)
   - 제1종 오류 최소화 필요

✅ 결론:
   "신약이 기존약보다 혈압 강하 효과가 유의하게 컸다
   (p = 0.002 < α = 0.01, d = 0.39).
   통계적으로 유의하지만 효과 크기는 작은 편이다.
   임상적 의미는 추가 평가 필요."

⚠️ 만약 α = 0.05였다면?
   → 여전히 유의 (p = 0.002 < 0.05)
   → 하지만 의학 연구는 더 엄격한 기준 요구
```

### 5.3 품질 관리: 공정 개선

**연구 질문**:
> "새 공정이 기존 공정보다 불량률을 낮추는가?"

**연구 설계**:
- 독립변수: 공정 (기존, 신규)
- 종속변수: 불량률 (%)
- 표본 크기: 각 500개 제품

**결과**:
```
기존: 5.2% (26/500)
신규: 3.4% (17/500)
z = 1.89, p = 0.029 (단측, less)
효과 크기: OR = 0.64 (불량률 36% 감소)
```

**해석**:
```
✅ 단측 검정 선택 이유:
   - 목표 명확: "개선" 입증
   - 불량률 증가는 애초에 채택 안 함
   - 검정력 증가 필요 (단측이 2배)

✅ α = 0.05 선택 이유:
   - 품질 관리 표준
   - 경영진 보고용

✅ 결론:
   "신규 공정이 불량률을 유의하게 낮췄다
   (p = 0.029, 단측). 불량률이 5.2%에서 3.4%로
   36% 감소했으며, 실무적으로 의미 있는 개선이다."

⚠️ 양측으로 했다면?
   → p = 0.058 (비유의, α = 0.05 기준)
   → 같은 데이터인데 검정력 부족으로 못 잡음!
   → 하지만 리스크: 만약 불량률이 증가했다면 탐지 못 함
```

### 5.4 탐색적 연구: 파일럿 연구

**연구 질문**:
> "새로운 사료 첨가제가 성장률에 영향을 주는가?" (예비 연구)

**연구 설계**:
- 독립변수: 첨가제 (무첨가, 첨가)
- 종속변수: 30일 후 체중 증가량 (g)
- 표본 크기: 각 15마리 (소규모 파일럿)

**결과**:
```
무첨가: 평균 45g, SD = 12g
첨가:   평균 52g, SD = 15g
t(28) = 1.45, p = 0.08 (양측)
Cohen's d = 0.51 (중간 효과)
```

**해석**:
```
✅ 양측 검정 선택 이유:
   - 처음 연구 (방향 모름)
   - 첨가제 부작용 가능성도 확인 필요

✅ α = 0.10 선택 이유:
   - 탐색적 연구 (파일럿)
   - 경향성 파악 목적
   - 본 연구 설계를 위한 사전 정보

✅ 결론:
   "첨가제가 성장률을 증가시키는 경향이 있었다
   (p = 0.08 < α = 0.10, d = 0.51).
   효과 크기는 중간 수준으로, 본 연구를 진행할 가치가 있다.
   더 큰 표본으로 재검증 필요."

⚠️ 만약 α = 0.05였다면?
   → p = 0.08 > 0.05 (비유의)
   → 효과가 있을 수도 있는데 놓침! (검정력 부족)
   → 하지만 파일럿에서는 경향성 파악이 목적
```

## 6. 실무 체크리스트

### 6.1 연구 시작 전

- [ ] 연구 질문 명확히 정의
- [ ] 양측/단측 검정 선택 (정당화 포함)
- [ ] 유의수준(α) 선택 (분야 관행 고려)
- [ ] 필요 표본 크기 계산 (검정력 분석)
- [ ] **사전 등록** (p-hacking 방지)

---

### 6.2 분석 중

- [ ] 데이터 정규성/등분산성 확인
- [ ] 적절한 통계 검정 선택
- [ ] p-value 계산
- [ ] 효과 크기 계산
- [ ] 신뢰구간 계산

---

### 6.3 결과 보고

- [ ] p-value 정확히 보고 (< 0.001 제외하고 정확한 값)
- [ ] 효과 크기 보고 (Cohen's d, r, η² 등)
- [ ] 신뢰구간 보고
- [ ] 양측/단측 명시
- [ ] α 값 명시
- [ ] 실질적 의미 해석 (효과 크기 기반)

---

### 6.4 흔한 실수 피하기

❌ **하지 말아야 할 것**:
- p-hacking (유의하게 나올 때까지 분석 반복)
- HARKing (결과 보고 가설 만들기)
- 결과 보고 α 변경 (사후 조작)
- p-value만 보고 (효과 크기 무시)
- "유의하지 않음" = "차이 없음" (잘못된 해석)

✅ **해야 할 것**:
- 사전 연구 계획 (pre-registration)
- p-value + 효과 크기 모두 보고
- 실질적 의미 고려
- 제한점 명시
- 재현성 확보 (데이터/코드 공개)
