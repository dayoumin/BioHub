
## 📋 사용자 설치 가이드

### 빠른 시작 (3단계)

#### Step 1: 파일 준비
- ZIP 파일 다운로드
- 압축 해제
- `deployment-package/` 폴더 열기

#### Step 2: 앱 실행
**Windows**:
```bash
start.bat 더블클릭
```

**macOS/Linux**:
```bash
./start.sh 실행
```

또는 직접:
```bash
cd statistical-app
python -m http.server 8000
```

#### Step 3: 브라우저 접속
```
http://localhost:8000
```

**소요 시간**: 5-10분

## ✨ 주요 기능

### 즉시 사용 가능 (설정 불필요)
- ✓ 기술 통계량 (평균, 표준편차 등)

- ✓ 상관분석, 주성분분석
- ✓ 시각화 (차트, 테이블)

- ✓ 분석 결과 해석
- ✓ 보고서 자동 생성
- ✓ PDF 문서 처리
