## 1. 해외 통계 도구 가격 현황 (2025년 기준)

> **참고**: 외산 소프트웨어는 환율 변동($1 = 1,300~1,400원)에 따라 원화 체감가 차이가 큼.
> 한국 대학은 KASS 등 대행사를 통해 공동 구매하므로, **재학 중 학생 체감가는 대부분 0원** (학교 포털 다운로드, 90%+).
> 단, 졸업/퇴직 후에는 개인 구매가 적용됨.

### 개인/학생

| 도구 | 가격 | 비고 |
|------|------|------|
| **SPSS** | $99~105/월 (구독), 한국 대행사 월 14~15만원 | 영구: Standard 400~500만원, Premium 1,000만원+ (모듈별 추가 구매) |
| **GraphPad Prism** | $142/년 (개인, 1인 1기기) | 영구: $3,200 (1년 업데이트), 최근 구독 전환 추세 |
| **SAS** | 무료 (OnDemand for Academics) | 학습 목적 한정, 클라우드 전용 |
| **Stata** | $48~$595 (영구, 에디션별) | 학생용 $48 (소규모 데이터) |
| **MedCalc** | €100+/년 | 임상 특화 |
| **R / jamovi / JASP** | 무료 | 설치 필요, 영어 전용 |
| **이 플랫폼** | **무료** (Pro $9.99/월) | 웹 기반, 한글, 졸업 후에도 동일 |

### 기관/기업

| 도구 | 가격 | 비고 |
|------|------|------|
| **SPSS** | Campus 연 ~$50,000 | 한국 기관 수천만원/년, 모듈 추가 시 기하급수적 증가 (Amos 등) |
| **GraphPad Prism** | Academic $500/년 (5~10명 연구실 단위 시작), Corporate $1,040/년 | 사용자당 |
| **SAS** | 한국 대학 1,365만~3,350만원+/년 | 엔터프라이즈 설치형 기준. 대기업 유지보수만 억 단위 |
| **Stata** | $1,395~$3,695/년 (기관용) | 에디션·사용자 수별 |
| **이 플랫폼** | **내부망 배포 비용 0** | 정적 HTML, 서버 불필요 |

### 시나리오 C: 개인 연구자/대학원생

> **현실**: 재학 중에는 KASS 등 대학 공동구매로 SPSS/SAS 체감가 0원 (학교 포털 다운로드).
> 하지만 **졸업 후** 개인 구매 시 비용이 급증하며, 도구 연속성이 끊김.

| 상황 | SPSS | GraphPad Prism | 이 플랫폼 |
|------|------|---------------|----------|
| **재학 중** | ₩0 (학교 라이선스) | ₩0 (학교 라이선스) | ₩0 (Free) |
| **졸업 후** | ~₩1,500,000/년 ($105/월) | ~₩200,000/년 ($142/년) | ₩0 (Free) 또는 ~₩160,000 (Pro) |

**핵심 가치**: 졸업/퇴직 후에도 동일 도구를 무료로 계속 사용 → vendor lock-in 없음.
재학 중에는 비용 절감보다 **학습 연속성**이 차별점.

## 5. 한국 시장 도입 전략

### 진입 경로

| 단계 | 타겟 | 전략 |
|------|------|------|
| 1 | 대학원생 | 무료 + 한글 → "Prism/SPSS 대신 이거 써봐" |
| 2 | 지도교수/PI | 학생이 쓰는 도구 → 랩 표준 채택 |
| 3 | 대학 기관 | SPSS Campus 라이선스 대체 → 연 수천만원 절감 |
| 4 | 정부 연구기관 | 내부망 정적 배포 → 보안 + 비용 0 |
| 5 | 학회/협회 | 학회 표준 분석 도구 → 회원 확산 |

### 기관 도입 시 제안 포인트

- **비용**: SPSS/SAS 라이선스 대비 90~100% 절감
- **보안**: 정적 HTML 배포, 데이터가 서버로 전송되지 않음 (Pyodide = 브라우저 내 실행)
- **접근성**: 설치 불필요, 브라우저만 있으면 됨
- **연속성**: 졸업/퇴직 후에도 동일 도구 무료 사용 (vendor lock-in 없음)
- **한글**: UI, 결과 해석, 도움말 모두 한글
