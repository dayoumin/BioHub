## 1. 미토콘드리아 DNA (mtDNA) 마커

### 1.1 COI (Cytochrome c Oxidase Subunit I) — 표준 바코드

> 동물 DNA 바코딩의 국제 표준. Hebert et al. (2003) 이래 가장 널리 사용.

| 항목 | 내용 |
|------|------|
| **단편 길이** | 표준 658 bp (Folmer region), 미니바코드 ~313 bp, 초소형 ~130 bp |
| **진화 속도** | 중간~빠름 |
| **종내 변이** | 일반적 <2% |
| **종간 변이** | 일반적 >3% (분류군별 차이 큼) |
| **범용 프라이머** | LCO1490/HCO2198 (Folmer 1994), mlCOIintF/jgHCO2198 (메타바코딩) |
| **주요 DB** | BOLD (1,700만 표본, 2,180만 공개 레코드, 130만+ 종), GenBank |

**강점**:
- 가장 큰 참조 DB (BOLD)
- 대부분 동물에서 잘 정립된 barcode gap
- 표준화된 프로토콜과 생물정보학 파이프라인

**한계**:
- 모계 유전만 반영 (교잡/유전자 침투 감지 불가)
- NUMTs (핵 위유전자) 공동 증폭 가능
- 일부 분류군에서 실패 (참치, 자포동물, 해면 등)

**실패 시 대안**:

| 상황 | 대안 마커 | 이유 |
|------|-----------|------|
| 참치류 (Thunnus) | D-loop/CR + ITS1 | COI 종간 변이 부족, CR은 10배 높은 변이 |
| 양서류 | 16S rRNA | 프라이머 범용성 우수, 더 큰 참조 DB |
| 자포동물/해면 | 16S rRNA | COI 진화 속도 너무 느림 |
| 동물플랑크톤 | 18S rRNA + COI | 미성숙개체 형태 판별 불가 |
| 가공식품 | 미니바코드 (127-314 bp) | 전체 658 bp 증폭 실패율 80% |
| eDNA | 12S rRNA (MiFish) | 짧은 단편 (170 bp)으로 분해된 DNA 검출 |

### 1.2 Cyt b (Cytochrome b) — 포유류 포렌식 표준

> COI 표준화 이전 척추동물 계통학의 주력 마커. 포유류 법의학 동정에서 여전히 최다 사용.

| 항목 | 내용 |
|------|------|
| **단편 길이** | 전체 ~1,140 bp, 법의학 300-500 bp, 미니 ~136 bp |
| **진화 속도** | 중간 (COI와 유사) |
| **범용 프라이머** | L14724/H15915 (Irwin 1991) — 포유류 전체 유전자 증폭 |
| **주요 DB** | GenBank (많은 척추동물에서 COI보다 풍부) |

**강점**:
- GenBank에서 포유류 참조 서열이 COI보다 풍부
- 분해된 DNA에서도 작동 (뼈, 법의학 시료)
- 식품 위조 판별에 확립된 방법

**한계**:
- BOLD 표준이 아님 (큐레이션된 DB 적음)
- 설치류/영장류에서 NUMTs 간섭
- 무척추동물에서 범용성 낮음

**실패 시 대안**: COI (더 넓은 DB), D-loop (더 높은 변이), RAG1 (핵 확인)

---

### 1.3 D-loop / Control Region (CR) — 집단 유전학의 핵심

> 미토콘드리아 비코딩 조절 영역. mtDNA에서 가장 빠르게 진화하는 영역.

| 항목 | 내용 |
|------|------|
| **단편 길이** | 전체 800-1,200 bp, 과변이 영역 (HVS-I/II) 200-400 bp |
| **진화 속도** | 빠름 — COI보다 ~10배 높은 뉴클레오티드 다양성 (참치 등) |
| **범용 프라이머** | 없음 — 분류군별 특이적 프라이머 필요 |
| **주요 DB** | GenBank |

**강점**:
- COI로 구분 불가한 근연종 해결 (참치, 연어 등)
- 종내 집단 구조 파악에 최적
- 보전 단위 설정, 이동 경로 추적

**한계**:
- 길이 이질성 (탠덤 반복)으로 정렬 어려움
- 범용 프라이머 없음
- 심화 계통학에 부적합 (포화 빠름)

**사용 시나리오**: COI로 종 구분 실패 시 첫 번째 대안 (특히 어류)

---

### 1.4 16S rRNA — 양서류 표준 + eDNA

> 미토콘드리아 대소단위 리보솜 RNA 유전자. 보존 영역과 변이 영역이 교대하여 범용 프라이머 설계에 유리.

| 항목 | 내용 |
|------|------|
| **단편 길이** | 전체 ~1,500 bp, 일반 ~500 bp, eDNA ~219-250 bp |
| **진화 속도** | 느림~중간 |
| **범용 프라이머** | 16Sar/16Sbr (Palumbi 1996), Berry_16S (eDNA) |
| **주요 DB** | GenBank (양서류 최대), NCBI |

**강점**:
- 양서류에서 COI보다 우수 (프라이머 범용성, 참조 DB 크기)
- 자포동물 동정에 최적 (해파리류)
- eDNA 메타바코딩에 적합한 짧은 앰플리콘

**한계**:
- 일부 곤충에서 COI보다 해상도 낮음
- 인델로 정렬 어려움
- 최근 분화 종에서 변이 부족

---

### 1.5 12S rRNA — eDNA 금표준

> 미토콘드리아 소단위 리보솜 RNA 유전자. 어류 eDNA 메타바코딩의 표준.

| 항목 | 내용 |
|------|------|
| **단편 길이** | 전체 ~950 bp, MiFish ~170 bp, Teleo ~65 bp, Riaz ~106 bp |
| **진화 속도** | 느림~중간 |
| **범용 프라이머** | MiFish-U (Miya 2015), Teleo (Valentini 2016), VertU (2023) |
| **주요 DB** | GenBank, MitoFish |

**강점**:
- 어류 eDNA 메타바코딩 금표준 (MiFish 프라이머)
- 짧은 앰플리콘 → 분해된 eDNA에 이상적
- MiFish + Teleo 조합 시 93.4% 종 커버리지

**한계**:
- 종 풍부 속에서 COI보다 해상도 낮음
- 프라이머 편향 (특정 종 우선 증폭)
- 일부 지역 참조 DB 불완전

## 2. 핵 DNA (nDNA) 마커

### 2.1 ITS (Internal Transcribed Spacer) — 진균 표준 + 동물 보조

> 리보솜 DNA 반복 단위의 스페이서 영역. 진균의 공식 바코드 (ITS2). 동물에서는 mtDNA 실패 시 보조 마커.

| 항목 | 내용 |
|------|------|
| **단편 길이** | ITS1: 200-700 bp, ITS2: 200-500 bp, 전체: 600-1,200 bp |
| **진화 속도** | 중간~빠름 (핵 마커 중) |
| **범용 프라이머** | ITS1/ITS4 (White 1990) — 진핵생물 범용 |
| **주요 DB** | GenBank, UNITE (진균) |

**강점**:
- 다복사 (게놈당 수백 개 반복) → 낮은 품질 DNA에서도 높은 감도
- 핵 마커로 양친 유전 반영 → 교잡 감지
- 참치 유전자 침투 확인 시 ITS1 필수

**한계**:
- 게놈 내 변이 (불완전 조화 진화)
- 동물 참조 DB가 BOLD보다 매우 작음

---

### 2.2 RAG1 (Recombination Activating Gene 1) — 척추동물 심화 계통학

> 척추동물 면역계 필수 유전자. 단일 복사 핵 유전자로 패밀리/목 수준 계통 분석에 사용.

| 항목 | 내용 |
|------|------|
| **단편 길이** | 전체 ~3,100 bp (단일 엑손), 일반 ~1,500 bp |
| **진화 속도** | 느림 — 인델 희귀, 전이 포화 미미 |
| **범용 프라이머** | 척추동물 범용 다수 존재 |
| **주요 DB** | GenBank |

**강점**:
- 단일 복사 → 이배체성/다복사 문제 없음
- 인트론 없음 → 정렬 용이
- mtDNA와 독립적 핵 계통 제공 → 교잡/유전자 침투 감지

**한계**:
- 최근 분화 종에서 변이 부족 (종 수준 동정에 부적합)
- 긴 단편 필요 (~1,500 bp) → 분해 시료/eDNA 부적합

---

### 2.3 SNPs (Single Nucleotide Polymorphisms) — 품종/계통 판별

> 게놈 전체의 단일 염기 변이. 집단 유전학과 품종 판별의 최신 표준.

| 항목 | 내용 |
|------|------|
| **분석 단위** | 수십~수천 개 로커스 (패널) 또는 수백만 (전장 게놈) |
| **진화 속도** | 로커스당 낮음 (~10⁻⁸/bp/세대), 게놈 규모로 보상 |
| **범용 프라이머** | 없음 — 종/그룹별 패널 개발 필요 |
| **주요 DB** | dbSNP, 종별 SNP DB |

**강점**:
- 집단 수준 다양성의 가장 정확한 추정
- 품종 판별 가능 (BovineSNP50: 53,218개 SNP → 95%+ 품종 분류 정확도)
- 게놈 전체 균등 분포

**한계**:
- 참조 게놈 또는 발굴 노력 필요 (초기 비용)
- 이대립 → 로커스당 정보량 microsatellite보다 적음
- 게놈 자원 없는 종에 적용 불가

---

### 2.4 Microsatellites (SSRs/STRs) — 집단 유전학 + 개체 식별

> 1-6 뉴클레오티드 짧은 탠덤 반복. 공우성, 고다형성. SNP 이전 집단 유전학의 주력.

| 항목 | 내용 |
|------|------|
| **단편 길이** | 반복 배열 10-300 bp, PCR 산물 100-400 bp |
| **진화 속도** | 매우 빠름 (10⁻²~10⁻⁶/로커스/세대) |
| **일반 사용** | 8-20 로커스/연구 |
| **주요 DB** | 종별 논문 (중앙 DB 없음) |

**강점**:
- 로커스당 최고 다형성
- 참조 게놈 불필요 (전통적 분리 가능)
- 친자/혈연 분석, 개체 식별에 확립

**한계**:
- 동형동조 (크기 수렴) 문제
- 개발 노동 집약적
- SNP에 의해 대체되는 추세

## 3. 게놈 접근법

### 3.1 RAD-seq — 참조 게놈 없이 수천 SNP

> 제한효소로 게놈을 단편화하고 절단 부위 인접 영역을 시퀀싱. 참조 게놈 없이 수천 개 SNP 발굴.

| 항목 | 내용 |
|------|------|
| **로커스 수** | 수천~수만 개 |
| **변이체** | ddRAD, 2bRAD, ezRAD |
| **입력 요구** | 고품질 고분자량 DNA |

**강점**: 형태/mtDNA로 해결 불가한 종 복합체 해결, 교잡대 분석
**한계**: eDNA/분해 시료 부적합, 연구실 간 비교 어려움

### 3.2 eDNA 메타바코딩 — 환경 군집 조사

> 환경 시료(물, 토양, 공기)에서 추출한 DNA를 범용 프라이머로 증폭 → HTS로 군집 내 모든 종 동시 동정.

| 대상 분류군 | 1차 마커 | 프라이머 | 앰플리콘 |
|-------------|----------|----------|----------|
| 어류 | 12S rRNA | MiFish-U | ~170 bp |
| 포유류 | 12S rRNA | MiMammal | ~170 bp |
| 조류 | 12S rRNA | MiBird | ~170 bp |
| 양서류 | 12S + 16S | 다수 | 150-250 bp |
| 전체 척추동물 | 12S | VertU | 200-250 bp |
| 무척추동물/곤충 | COI | mlCOIintF/jgHCO | 313 bp |

## 4. 종 경계 판별 방법 (Species Delimitation)

| 방법 | 유형 | 입력 | 교정 필요? | 경향 | 최적 용도 |
|------|------|------|-----------|------|-----------|
| **BIN** (BOLD) | 거리 기반 | COI 서열 | 아니오 | 보통 | 표준 조사 |
| **ABGD** | 거리 기반 | 정렬 서열 | 사전 P값 | 과소 분할 | 충분히 샘플링된 그룹 |
| **ASAP** | 거리 기반 | 거리 행렬 | 아니오 | 보통 | 탐색적, 사전 정보 없음 |
| **GMYC** | 계통수 기반 | 초시계 계통수 | 예 | 과다 분할 | 시간 교정 데이터셋 |
| **bPTP** | 계통수 기반 | 계통수 | 아니오 | 보통 | 표준 계통 데이터 |

**모범 사례**: 복수 방법 사용 → 일치 결과에 높은 신뢰도. ABGD + ASAP + bPTP 일치 시 가장 강력.

## 5. 마커 선택 의사결정 트리

```
시작: 대상 분류군?

곤충/절지동물 ──→ COI (1차)
  └─ 실패? → ITS2 또는 18S

어류 ──→ COI (동정) 또는 12S (eDNA)
  └─ COI 실패 (참치 등)? → D-loop/CR + ITS1
  └─ eDNA 조사? → MiFish (12S) + Teleo, 16S 추가

포유류 ──→ COI 또는 Cyt b
  └─ 법의학/분해? → Cyt b 미니바코드
  └─ 품종 판별? → SNP 패널 또는 microsatellite
  └─ 집단 수준? → D-loop/CR

양서류 ──→ 16S rRNA (1차) + COI
  └─ eDNA? → 12S + 16S

조류 ──→ COI (1차)
  └─ 근연종? → Cyt b + CR
  └─ 최근 방산? → 핵 인트론 (FIB5, ODC)

자포동물/해면 ──→ 16S rRNA (COI 아님)

기생충/편형동물 ──→ COI + ITS2

종 복합체/은밀종 ──→ 다중 마커 + RAD-seq

집단 유전학 ──→ microsatellite 또는 SNP 패널

군집 조사 ──→ eDNA 메타바코딩 (12S/16S/COI)
```

### 입력 → 분석 → 보고서 파이프라인

```
1. 서열 입력 (FASTA)
2. 품질 검사 (길이, GC 함량, 모호 염기)
3. DB 검색 (BOLD → NCBI BLAST 순차)
4. 유사도 매칭 + BIN 할당
5. 실패 시 대안 마커 제안 (의사결정 트리 기반)
6. 보고서 자동 생성 (논문 삽입 가능 형식)
```

### 보고서 포함 항목 (표준)

- 사용 마커 및 프라이머 정보
- 서열 품질 통계 (길이, GC%, 모호 염기 수)
- 최상위 매칭 종 + 유사도 (%)
- BIN 번호 (해당 시)
- 유전 거리 (K2P)
- 분석 실패 시: 실패 원인 + 권장 대안 마커 + 근거
